## v8.0.0 — February 2026

**Major Release — Complete Voice AI Platform**

### New Features
- **194 API endpoints** across 25 categories
- **In-call Chat** — Send text messages during active voice calls
- **Post-call Surveys** — Configurable survey templates with voice delivery
- **Predictive Analytics** — Volume forecasting, escalation risk, churn prediction
- **Call Tagging** — Manual and auto-tagging with keyword rules
- **Hold Music** — Upload, manage, and play hold music during calls
- **Escalation Rules** — Condition-based auto-escalation engine
- **Call Masking** — Number privacy with masked sessions
- **Voice Biometrics** — Speaker enrollment and verification
- **GDPR Compliance** — SAR, erasure, consent management

### Improvements
- LLM failover chain (Groq → OpenAI → Anthropic)
- Enhanced batch calling with up to 1000 concurrent calls
- BullMQ-based queue system for reliable job processing
- Circuit breaker pattern for upstream provider resilience
- Comprehensive PII detection (15+ types)

### Infrastructure
- Multi-tenant architecture with full data isolation
- Redis-backed state management and caching
- MongoDB with per-tenant collections
- PM2 cluster mode with auto-restart
