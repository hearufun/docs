## Prerequisites

Before you begin, ensure you have:

- A TalkOS account (or create one below)
- A Plivo or Exotel account with API credentials
- At least one phone number purchased from your voice provider

## Step 1: Create Your Tenant

Every organization using TalkOS is a "tenant". Create your tenant to get API credentials:

<CodeGroup>

```bash cURL
curl -X POST https://api.talkos.io/api/tenants/create \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Corporation",
    "email": "admin@acme.com",
    "plan": "starter",
    "config": {
      "defaultProvider": "plivo",
      "plivo": {
        "authId": "YOUR_PLIVO_AUTH_ID",
        "authToken": "YOUR_PLIVO_AUTH_TOKEN"
      }
    }
  }'
```

```javascript Node.js
const response = await fetch('https://api.talkos.io/api/tenants/create', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: 'Acme Corporation',
    email: 'admin@acme.com',
    plan: 'starter',
    config: {
      defaultProvider: 'plivo',
      plivo: {
        authId: 'YOUR_PLIVO_AUTH_ID',
        authToken: 'YOUR_PLIVO_AUTH_TOKEN'
      }
    }
  })
});
const data = await response.json();
console.log('API Key:', data.tenant.apiKey);
```

```python Python
import requests

response = requests.post(
    'https://api.talkos.io/api/tenants/create',
    json={
        'name': 'Acme Corporation',
        'email': 'admin@acme.com',
        'plan': 'starter',
        'config': {
            'defaultProvider': 'plivo',
            'plivo': {
                'authId': 'YOUR_PLIVO_AUTH_ID',
                'authToken': 'YOUR_PLIVO_AUTH_TOKEN'
            }
        }
    }
)
data = response.json()
print('API Key:', data['tenant']['apiKey'])
```

</CodeGroup>

<Note>
Save the `apiKey` and `tenantId` from the response - you'll need these for all API calls.
</Note>

## Step 2: Configure Your Voice Provider

### Option A: Plivo Setup

1. Log into your [Plivo Console](https://console.plivo.com)
2. Go to **Voice** → **Applications** → **Create New Application**
3. Configure these webhooks:

| Field | Value |
|-------|-------|
| Answer URL | `https://api.talkos.io/api/calls/answer` |
| Answer Method | POST |
| Hangup URL | `https://api.talkos.io/api/calls/hangup` |
| Hangup Method | POST |

4. Assign your phone numbers to this application

### Option B: Exotel Setup

1. Log into your [Exotel Dashboard](https://my.exotel.com)
2. Navigate to **App Store** → **Create App**
3. Set the passthru applet with URL: `https://api.talkos.io/api/calls/exotel/status-callback`
4. Assign your ExoPhone to this app

## Step 3: Buy a Phone Number

Purchase a phone number through TalkOS (it automatically configures the provider):

```bash
curl -X POST https://api.talkos.io/api/numbers/buy \
  -H "x-api-key: tk_live_xxxxxxxxxxxxxxxxxxxx" \
  -H "x-tenant-id: tenant_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "dialerProvider": "plivo",
    "country": "IN",
    "type": "local",
    "areaCode": "22",
    "direction": "both"
  }'
```

### Response

```json
{
  "success": true,
  "phoneNumber": {
    "number": "+912212345678",
    "dialerProvider": "plivo",
    "status": "active",
    "direction": "both",
    "tenantId": "tenant_abc123"
  }
}
```

## Step 4: Make Your First Call

Now you're ready to make an AI-powered call:

```bash
curl -X POST https://api.talkos.io/api/calls/start \
  -H "x-api-key: tk_live_xxxxxxxxxxxxxxxxxxxx" \
  -H "x-tenant-id: tenant_abc123" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "+919876543210",
    "from": "+912212345678",
    "prompt": "You are Aria, a friendly customer success representative from Acme Corporation. Your goal is to check in with the customer about their recent purchase and see if they have any questions. Be helpful, concise, and professional.",
    "greeting": "Hello! This is Aria from Acme Corporation. I am calling to check in on your recent purchase. How has your experience been so far?",
    "webhookUrl": "https://your-server.com/call-completed",
    "maxDuration": 300,
    "language": "en-IN"
  }'
```

### Response

```json
{
  "success": true,
  "callUuid": "call_xyz789",
  "jobId": "job_123",
  "status": "queued",
  "to": "+919876543210",
  "from": "+912212345678",
  "estimatedStartTime": "2025-02-18T00:00:05.000Z"
}
```

## Step 5: Check Call Status

Monitor the call progress:

```bash
curl -X GET "https://api.talkos.io/api/calls/call_xyz789/status" \
  -H "x-api-key: tk_live_xxxxxxxxxxxxxxxxxxxx" \
  -H "x-tenant-id: tenant_abc123"
```

### Response

```json
{
  "success": true,
  "call": {
    "callUuid": "call_xyz789",
    "status": "completed",
    "to": "+919876543210",
    "from": "+912212345678",
    "duration": 127,
    "startedAt": "2025-02-18T00:00:06.000Z",
    "endedAt": "2025-02-18T00:02:13.000Z",
    "sentiment": "positive",
    "summary": "Customer expressed satisfaction with their purchase. No issues reported. Scheduled a follow-up demo for advanced features."
  }
}
```

  "to": "+919876543210",
  "from": "+912212345678",
  "duration": 127,
  "status": "completed",
  "hangupCause": "normal_clearing",
  "transcript": [
    { "role": "assistant", "content": "Hello! This is Aria from Acme Corporation...", "timestamp": "2025-02-18T00:00:06.000Z" },
    { "role": "user", "content": "Hi Aria! Yes, I received it yesterday...", "timestamp": "2025-02-18T00:00:15.000Z" }
