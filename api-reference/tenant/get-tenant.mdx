# Get Tenant

Retrieves the complete details of an existing tenant, including their configuration, plan details, usage statistics, and status.

## Endpoint

```
GET https://api.talkos.io/api/tenants/:tenantId
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your tenant API key or admin API key.
  
  **Example:** `tk_live_abc123xyz789`
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  The tenant identifier. Usually the same as the path parameter for self-lookup.
  
  **Example:** `tenant_01HXYZ123456`
</ParamField>

## Path Parameters

<ParamField path="tenantId" type="string" required>
  The unique identifier of the tenant to retrieve.
  
  **Format:** Starts with `tenant_` followed by alphanumeric characters
  
  **Example:** `tenant_01HXYZ123456`
</ParamField>

## Query Parameters

<ParamField query="include" type="string">
  Comma-separated list of additional data to include in the response.
  
  **Allowed values:**
  - `usage` - Include current period usage statistics
  - `limits` - Include rate limit details
  - `features` - Include enabled features list
  - `all` - Include all additional data
  
  **Example:** `usage,limits`
</ParamField>

## Response

<ResponseField name="success" type="boolean">

</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.talkos.io/api/tenants/tenant_01HXYZ123456?include=usage,limits,features" \
  -H "x-api-key: tk_live_abc123xyz789" \
  -H "x-tenant-id: tenant_01HXYZ123456"
```

```javascript Node.js
const response = await fetch(
  'https://api.talkos.io/api/tenants/tenant_01HXYZ123456?include=usage,limits,features',
  {
    method: 'GET',
    headers: {
      'x-api-key': 'tk_live_abc123xyz789',
      'x-tenant-id': 'tenant_01HXYZ123456'
    }
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/tenants/tenant_01HXYZ123456',
    params={'include': 'usage,limits,features'},
    headers={
        'x-api-key': 'tk_live_abc123xyz789',
        'x-tenant-id': 'tenant_01HXYZ123456'
    }
)

print(response.json())
```

</CodeGroup>

## Notes

<Info>
  Tenants can only access their own tenant data unless using an admin API key with cross-tenant permissions.
</Info>

<Tip>
  Use the `include` query parameter to fetch only the data you need. This improves response time and reduces payload size.
</Tip>

<Warning>
  Sensitive configuration values like API tokens are always masked in the response for security.
</Warning>
