# Get Usage

Retrieves detailed usage statistics for a tenant, including call metrics, API usage, and resource consumption. This data is useful for monitoring usage, billing reconciliation, and capacity planning.

## Endpoint

```
GET https://api.talkos.io/api/tenants/:tenantId/usage
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your tenant API key.
  
  **Example:** `tk_live_abc123xyz789`
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  The tenant identifier.
  
  **Example:** `tenant_01HXYZ123456`
</ParamField>

## Path Parameters

<ParamField path="tenantId" type="string" required>
  The unique identifier of the tenant.
  
  **Example:** `tenant_01HXYZ123456`
</ParamField>

## Query Parameters

<ParamField query="period" type="string" default="current_month">
  The time period for usage data.
  
  **Allowed values:**
  - `today` - Current day (UTC)
  - `yesterday` - Previous day (UTC)
  - `current_week` - Current week (Monday-Sunday)
  - `last_week` - Previous week
  - `current_month` - Current calendar month
  - `last_month` - Previous calendar month
  - `custom` - Custom date range (requires `startDate` and `endDate`)
  
  **Example:** `current_month`
</ParamField>

<ParamField query="startDate" type="string">
  Start date for custom period. Required if `period=custom`.
  
  **Format:** ISO 8601 date (`YYYY-MM-DD`)
  
  **Example:** `2026-02-01`
</ParamField>

<ParamField query="endDate" type="string">
  End date for custom period. Required if `period=custom`.
  
  **Format:** ISO 8601 date (`YYYY-MM-DD`)
  
  **Example:** `2026-02-18`
</ParamField>

<ParamField query="granularity" type="string" default="total">
  How to break down the usage data.
  
  **Allowed values:**
  - `total` - Single aggregated total
  - `daily` - Day-by-day breakdown
  - `hourly` - Hour-by-hour breakdown (only for today/yesterday)
  
  **Example:** `daily`
</ParamField>

<ParamField query="include" type="string">
  Comma-separated list of additional data to include.
  
  **Allowed values:**
  - `breakdown` - Include per-feature breakdown
  - `comparison` - Include comparison with previous period
  - `projections` - Include projected usage for the period
  - `limits` - Include current plan limits
  - `all` - Include everything
  
  **Example:** `breakdown,limits`
</ParamField>

## Notes

<Info>
  **Data Freshness**: Usage data is updated in near real-time. There may be a delay of up to 5 minutes for the most recent data.
</Info>

<Tip>
  **Billing Reconciliation**: For billing purposes, use `period=last_month` to get the complete usage for the previous billing cycle.
</Tip>

<Warning>
  **High Utilization Alert**: If `utilizationPercent` for concurrent calls is consistently above 80%, consider upgrading your plan to avoid hitting limits during peak times.
</Warning>

<Note>
  **Historical Data**: Usage data is retained for 13 months. Queries for older data will return an error.
</Note>

## Use Cases

### Monitoring Dashboard

Fetch real-time usage for dashboard displays:

```bash
curl "https://api.talkos.io/api/tenants/{tenantId}/usage?period=today&granularity=hourly&include=limits"
```

### Monthly Billing Report

Generate end-of-month usage report:

```bash
curl "https://api.talkos.io/api/tenants/{tenantId}/usage?period=last_month&include=breakdown,all"
```

### Capacity Planning

Check projections and utilization:

```bash
curl "https://api.talkos.io/api/tenants/{tenantId}/usage?period=current_month&include=projections,limits"
```
