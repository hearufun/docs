# Get Dashboard Data

Retrieve comprehensive real-time dashboard data for supervisors, including active calls, performance statistics, critical alerts, and geographic heatmap data.

## Endpoint

```
GET /api/dashboard
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="timestamp" type="number">
  Unix timestamp (milliseconds) when the data was generated
</ResponseField>

<ResponseField name="dashboard" type="object">
  Dashboard data object containing:

  <Expandable title="dashboard properties">
    <ResponseField name="calls" type="array">
      Array of active call objects:
      - `callUuid` (string): Unique call identifier
      - `agentId` (string): Agent handling the call
      - `agentName` (string): Agent display name
      - `customerPhone` (string): Customer phone number (masked)
      - `status` (string): Call status (`ringing`, `in-progress`, `on-hold`)
      - `duration` (number): Current call duration in seconds
      - `sentiment` (string): Real-time sentiment (`positive`, `neutral`, `negative`)
      - `sentimentScore` (number): Sentiment score (-1 to 1)
      - `startTime` (string): ISO 8601 timestamp
    </ResponseField>

    <ResponseField name="stats" type="object">
      Real-time statistics:
      - `activeCalls` (number): Number of calls currently in progress
      - `waitingCalls` (number): Calls waiting in queue
      - `averageWaitTime` (number): Average wait time in seconds
      - `averageHandleTime` (number): Average handle time in seconds
      - `callsToday` (number): Total calls handled today
      - `agentsOnline` (number): Number of agents currently online
      - `agentsBusy` (number): Number of agents on calls
      - `agentsAvailable` (number): Number of agents available
      - `serviceLevel` (number): Service level percentage
    </ResponseField>

    <ResponseField name="critical" type="array">
      Array of critical call alerts:
      - `callUuid` (string): Call identifier
      - `alertType` (string): Type of alert (`negative_sentiment`, `long_hold`, `escalation_requested`)
      - `severity` (string): Alert severity (`warning`, `critical`)
      - `message` (string): Human-readable alert message
      - `timestamp` (string): When alert was triggered
    </ResponseField>

    <ResponseField name="heatmap" type="array">
      Geographic distribution data:
      - `region` (string): Region/state code
      - `callCount` (number): Number of calls from region
      - `avgSentiment` (number): Average sentiment score
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.talkos.io/api/dashboard" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript Node.js
const response = await fetch('https://api.talkos.io/api/dashboard', {
  method: 'GET',
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
console.log(`Active calls: ${data.dashboard.stats.activeCalls}`);
console.log(`Critical alerts: ${data.dashboard.critical.length}`);
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/dashboard',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    }
)

data = response.json()
print(f"Active calls: {data['dashboard']['stats']['activeCalls']}")
print(f"Agents online: {data['dashboard']['stats']['agentsOnline']}")
```
</CodeGroup>

## Error Responses

| Status | Error | Description |
|--------|-------|-------------|
| 401 | Unauthorized | Invalid or missing API key |
| 403 | Forbidden | API key does not have supervisor dashboard access |
| 500 | Internal server error | Server-side error occurred |

<Tip>
  For real-time updates, use the WebSocket connection at `wss://api.talkos.io/ws/supervisor` instead of polling this endpoint.
</Tip>
