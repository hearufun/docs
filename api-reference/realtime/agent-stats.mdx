# Get Agent Statistics

Retrieve real-time performance statistics for all agents, including call distribution, average handle times, and availability status.

## Endpoint

```
GET /api/agent-stats
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="timestamp" type="number">
  Unix timestamp (milliseconds) when the data was generated
</ResponseField>

<ResponseField name="agentStats" type="array">
  Array of agent statistics objects:

  <Expandable title="agent properties">
    <ResponseField name="agentId" type="string">
      Unique agent identifier
    </ResponseField>

    <ResponseField name="name" type="string">
      Agent display name
    </ResponseField>

    <ResponseField name="status" type="string">
      Current agent status:
      - `online` - Available for calls
      - `busy` - Currently on a call
      - `away` - Temporarily unavailable
      - `offline` - Not logged in
    </ResponseField>

    <ResponseField name="currentCall" type="object">
      Current call details (null if not on call):
      - `callUuid` (string): Call identifier
      - `duration` (number): Call duration in seconds
      - `customerPhone` (string): Customer phone (masked)
      - `sentiment` (string): Current call sentiment
    </ResponseField>

    <ResponseField name="todayStats" type="object">
      Today's performance metrics:
      - `callsHandled` (number): Number of calls handled today
      - `totalTalkTime` (number): Total talk time in seconds
      - `averageHandleTime` (number): Average handle time in seconds
      - `averageSentiment` (number): Average sentiment score (-1 to 1)
      - `transferRate` (number): Percentage of calls transferred
      - `firstCallResolution` (number): FCR percentage
    </ResponseField>

    <ResponseField name="weekStats" type="object">
      This week's performance metrics:
      - `callsHandled` (number): Total calls this week
      - `averageHandleTime` (number): Average handle time
      - `customerSatisfaction` (number): CSAT score (1-5)
    </ResponseField>

    <ResponseField name="skills" type="array">
      List of agent skills/queues assigned
    </ResponseField>

    <ResponseField name="lastActivity" type="string">
      ISO 8601 timestamp of last activity
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.talkos.io/api/agent-stats" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript Node.js
const response = await fetch('https://api.talkos.io/api/agent-stats', {
  method: 'GET',
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
data.agentStats.forEach(agent => {
  console.log(`${agent.name}: ${agent.status} - ${agent.todayStats.callsHandled} calls`);
});
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/agent-stats',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    }
)

data = response.json()
for agent in data['agentStats']:
    print(f"{agent['name']}: {agent['status']} - {agent['todayStats']['callsHandled']} calls")
```
</CodeGroup>

## Error Responses

| Status | Error | Description |
|--------|-------|-------------|
| 401 | Unauthorized | Invalid or missing API key |
| 403 | Forbidden | API key does not have supervisor access |
| 500 | Internal server error | Server-side error occurred |
