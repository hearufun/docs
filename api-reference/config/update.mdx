---
title: "Update System Config"
api: "PUT https://api.talkos.io/api/config"
description: "Update server configuration fields"
---

Update server configuration fields. This is a partial update - only send the fields you want to change. Changes will propagate to all workers within 10 seconds.

## Request Headers

<ParamField header="x-updated-by" type="string">
  Identifier for who made the change (for audit purposes).
</ParamField>

<RequestExample>
```bash cURL (Change LLM Provider)
curl -X PUT https://api.talkos.io/api/config \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-api-key" \
  -H "x-updated-by: admin@example.com" \
  -d '{
    "llmProvider": "openai"
  }'
```

```bash cURL (Update Voice)
curl -X PUT https://api.talkos.io/api/config \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-api-key" \
  -d '{
    "elevenlabs": {
      "voiceId": "new-voice-id"
    }
  }'
```

```bash cURL (Update Multiple Fields)
curl -X PUT https://api.talkos.io/api/config \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-api-key" \
  -d '{
    "llmProvider": "groq",
    "timeouts": {
      "llm": 15000
    },
    "defaults": {
      "maxDuration": 3600
    }
  }'
```
</RequestExample>

## Notes

- Immutable fields (`_id`, `__v`, `createdAt`) cannot be updated
- The `version` field auto-increments with each update
- Changes trigger immediate reload on the current worker and propagate to others via sync
