---
title: "Get System Config"
api: "GET https://api.talkos.io/api/config"
description: "View the current server configuration"
---

Retrieve the current server configuration. API keys are masked for security.

<RequestExample>
```bash cURL
curl -X GET https://api.talkos.io/api/config \
  -H "x-api-key: your-api-key"
```
</RequestExample>

## Notes

- API keys and sensitive credentials are masked in the response
- The `syncStatus` object shows the health of the config synchronization service
- The `version` field increments with each config update
