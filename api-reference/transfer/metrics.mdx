---
title: 'Get Transfer Metrics'
api: 'GET https://api.talkos.io/api/transfer/metrics'
description: 'Retrieve call transfer performance metrics and analytics'
---

## Overview

Retrieves comprehensive metrics and analytics for call transfers, including success rates, average times, agent performance, and trend data. Use these metrics to optimize your transfer workflows and improve customer experience.

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Query Parameters

<ParamField query="startDate" type="string">
  Start date for metrics period (ISO 8601 format). Defaults to 7 days ago.
</ParamField>

<ParamField query="endDate" type="string">
  End date for metrics period (ISO 8601 format). Defaults to now.
</ParamField>

<ParamField query="granularity" type="string" default="daily">
  Time granularity for trend data: `hourly`, `daily`, `weekly`, `monthly`
</ParamField>

<ParamField query="agentId" type="string">
  Filter metrics for a specific agent
</ParamField>

<ParamField query="transferType" type="string">
  Filter by transfer type: `cold`, `warm`, `blind`
</ParamField>

<ParamField query="includeAgentBreakdown" type="boolean" default="false">
  Include per-agent performance breakdown
</ParamField>

<ParamField query="includeReasonBreakdown" type="boolean" default="false">
  Include breakdown by transfer reasons
</ParamField>

<ParamField query="includeTrends" type="boolean" default="true">
  Include historical trend data
</ParamField>

<RequestExample>
```bash cURL - Basic Metrics
curl -X GET "https://api.talkos.io/api/transfer/metrics" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL - With Date Range
curl -X GET "https://api.talkos.io/api/transfer/metrics?startDate=2026-02-01T00:00:00Z&endDate=2026-02-18T23:59:59Z&granularity=daily" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL - Agent-Specific
curl -X GET "https://api.talkos.io/api/transfer/metrics?agentId=agent_789xyz&startDate=2026-02-11T00:00:00Z" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL - Full Breakdown
curl -X GET "https://api.talkos.io/api/transfer/metrics?includeAgentBreakdown=true&includeReasonBreakdown=true&includeTrends=true" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL - By Transfer Type
curl -X GET "https://api.talkos.io/api/transfer/metrics?transferType=warm&granularity=hourly" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```
</RequestExample>

## Key Performance Indicators

| Metric | Description | Healthy Range |
|--------|-------------|---------------|
| Success Rate | % of transfers completed | > 85% |
| Average Accept Time | Seconds to accept | < 15 seconds |
| Average Hold Time | Caller wait time | < 30 seconds |
| Average Handle Time | Agent call duration | Varies by type |
| Accept Rate (per agent) | Agent's acceptance % | > 90% |

## Error Codes

| Code | Description |
|------|-------------|
| `INVALID_DATE_RANGE` | Start date must be before end date |
| `DATE_RANGE_TOO_LARGE` | Maximum range is 90 days |
| `INVALID_GRANULARITY` | Granularity not recognized |
| `AGENT_NOT_FOUND` | Specified agent ID does not exist |
| `INVALID_TRANSFER_TYPE` | Transfer type not recognized |
