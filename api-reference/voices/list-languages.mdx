---
title: "List Languages"
api: "GET https://api.talkos.io/api/languages"
description: "Retrieve all supported languages for voice synthesis"
---

## Overview

Get a list of all languages and locales supported by TalkOS voice synthesis. Use this to determine available language options when creating voices or configuring AI agents.

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request completed successfully
</ResponseField>

<ResponseField name="languages" type="array">
  Array of supported languages
  <Expandable title="language object">
    <ResponseField name="code" type="string">
      ISO language code (e.g., `en-US`, `es-ES`)
    </ResponseField>
    <ResponseField name="name" type="string">
      Human-readable language name
    </ResponseField>
    <ResponseField name="nativeName" type="string">
      Language name in native script
    </ResponseField>
    <ResponseField name="region" type="string">
      Geographic region or variant
    </ResponseField>
    <ResponseField name="voiceCount" type="number">
      Number of preset voices available for this language
    </ResponseField>
    <ResponseField name="cloningSupported" type="boolean">
      Whether voice cloning is supported for this language
    </ResponseField>
    <ResponseField name="realtimeSupported" type="boolean">
      Whether real-time synthesis is supported (low latency)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="totalCount" type="number">
  Total number of supported languages
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/languages" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```
</RequestExample>

## Language Capabilities

| Feature | Description |
|---------|-------------|
| `cloningSupported` | Voice cloning is available using audio samples in this language |
| `realtimeSupported` | Low-latency synthesis (&lt;200ms) available for live conversations |

## Usage Notes

<Tip>
When creating a cloned voice, use a sample recorded in the target language for best results. Cross-language cloning may produce unpredictable accents.
</Tip>

<Note>
Some languages support multiple regional variants. Choose the variant that best matches your target audience for natural pronunciation.
</Note>
