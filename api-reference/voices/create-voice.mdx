## Overview

Create a custom voice by cloning from audio samples. Custom voices allow your AI agents to speak with a consistent, branded voice identity. Voice cloning uses ElevenLabs' advanced voice cloning technology to generate natural-sounding speech based on your audio samples.

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  Unique name for your custom voice. Must be unique within your tenant.
</ParamField>

<ParamField body="description" type="string">
  Optional description of the voice (e.g., "Professional sales agent", "Friendly support")
</ParamField>

<ParamField body="files" type="array" required>
  Array of audio file objects. Minimum 1 file required.
  <Expandable title="File object properties">
    <ParamField body="data" type="string">
      Base64-encoded audio file. Supported formats: MP3, WAV, M4A, FLAC
    </ParamField>
    <ParamField body="filename" type="string">
      Original filename (e.g., "sample1.wav")
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="labels" type="array">
  Optional labels to categorize the voice (e.g., ["accent:british", "gender:female"])
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the voice was cloned successfully
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="voiceId" type="string">
      Unique ElevenLabs voice ID (use this in text-to-speech calls)
    </ResponseField>
    <ResponseField name="name" type="string">
      The voice name you provided
    </ResponseField>
  </Expandable>
</ResponseField>

## Audio Sample Requirements

<Info>
Voice cloning works best with high-quality audio samples:
- **Duration:** 30 seconds to 5 minutes (single speaker)
- **Quality:** Clear speech, minimal background noise
- **Format:** MP3, WAV, M4A, or FLAC
- **Sample Rate:** 44.1kHz or higher recommended
- **Speaker:** Single consistent speaker for best results
</Info>

<Tip>
**For best results:**
- Use natural, conversational speech with varied intonation
- Include diverse phonemes and word endings
- Avoid monotone or artificial-sounding speech
- Multiple samples improve cloning quality
</Tip>

```javascript Node.js
const axios = require('axios');
const fs = require('fs');

// Read and encode audio file
const audioFile = fs.readFileSync('sample.wav');
const base64Audio = audioFile.toString('base64');

const response = await axios.post('https://api.talkos.io/api/voices', {
  name: 'Emma Support',
  description: 'Professional support agent',
  files: [
    {
      data: base64Audio,
      filename: 'sample.wav'
    }
  ],
  labels: ['accent:american']
}, {
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id',
    'Content-Type': 'application/json'
  }
});

console.log('Voice cloned:', response.data.data.voiceId);
```

```python Python
import base64
import requests
import json

# Read and encode audio file
with open('sample.wav', 'rb') as f:
    audio_bytes = f.read()
    base64_audio = base64.b64encode(audio_bytes).decode('utf-8')

response = requests.post(
    'https://api.talkos.io/api/voices',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id',
        'Content-Type': 'application/json'
    },
    json={
        'name': 'Emma Support',
        'description': 'Professional support agent',
        'files': [
            {
                'data': base64_audio,
                'filename': 'sample.wav'
            }
        ],
        'labels': ['accent:american', 'tone:professional']
    }
)

print('Voice ID:', response.json()['data']['voiceId'])
```

## Using Cloned Voices

After cloning, use your `voiceId` in any text-to-speech call:

```bash
curl -X POST https://api.talkos.io/api/calls/start \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id" \
  -H "Content-Type: application/json" \
  -d '{
    "phoneNumber": "+15551234567",
    "message": "Hello, this is Emma speaking!",
    "voiceId": "JBFqnCBsd6RMkjVDRZzb"
  }'
```
