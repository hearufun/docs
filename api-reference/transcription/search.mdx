# Search Transcripts

Search through your call transcripts using full-text search queries and apply date range filters to find specific conversations.

## Endpoint

```
GET /api/transcripts/search
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Query Parameters

<ParamField query="q" type="string">
  Search query string. Searches across transcript content, speaker names, and metadata. Leave empty to return all transcripts.
</ParamField>

<ParamField query="startDate" type="string">
  Filter results from this date (ISO 8601 format). Example: `2026-01-01T00:00:00Z`
</ParamField>

<ParamField query="endDate" type="string">
  Filter results until this date (ISO 8601 format). Example: `2026-02-18T23:59:59Z`
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Maximum number of results to return (1-100)
</ParamField>

<ParamField query="skip" type="integer" default="0">
  Number of results to skip for pagination
</ParamField>

<ParamField query="sentiment" type="string">
  Filter by sentiment analysis result: `positive`, `neutral`, or `negative`
</ParamField>

<ParamField query="language" type="string">
  Filter by transcript language (ISO 639-1 code). Example: `en`, `es`, `hi`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  Search results object containing:

  <Expandable title="data properties">
    <ResponseField name="results" type="array">
      Array of matching transcript objects:
      - `callUuid` (string): Unique call identifier
      - `transcript` (array): Array of transcript segments
      - `duration` (number): Call duration in seconds
      - `language` (string): Detected language
      - `sentiment` (string): Overall sentiment
      - `createdAt` (string): ISO 8601 timestamp
      - `highlights` (array): Matched text snippets with context
    </ResponseField>

    <ResponseField name="total" type="integer">
      Total number of matching transcripts
    </ResponseField>

    <ResponseField name="limit" type="integer">
      Applied limit value
    </ResponseField>

    <ResponseField name="skip" type="integer">
      Applied skip value
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.talkos.io/api/transcripts/search?q=refund&startDate=2026-01-01T00:00:00Z&endDate=2026-02-18T23:59:59Z&limit=10" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript Node.js
const params = new URLSearchParams({
  q: 'refund',
  startDate: '2026-01-01T00:00:00Z',
  endDate: '2026-02-18T23:59:59Z',
  limit: '10'
});

const response = await fetch(
  `https://api.talkos.io/api/transcripts/search?${params}`,
  {
    method: 'GET',
    headers: {
      'x-api-key': 'your-api-key',
      'x-tenant-id': 'your-tenant-id'
    }
  }
);

const data = await response.json();
console.log(`Found ${data.data.total} transcripts`);
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/transcripts/search',
    params={
        'q': 'refund',
        'startDate': '2026-01-01T00:00:00Z',
        'endDate': '2026-02-18T23:59:59Z',
        'limit': 10
    },
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    }
)

data = response.json()
print(f"Found {data['data']['total']} transcripts")
```
</CodeGroup>

## Error Responses

| Status | Error | Description |
|--------|-------|-------------|
| 400 | Invalid date format | startDate or endDate is not valid ISO 8601 |
| 401 | Unauthorized | Invalid or missing API key |
| 500 | Internal server error | Server-side error occurred |
