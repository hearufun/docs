---
title: "List Bot Numbers"
api: "GET https://api.talkos.io/api/inbound/bot-numbers"
description: "Retrieve all phone numbers configured for inbound AI handling."
---

Get a list of all phone numbers assigned to your tenant that are configured for inbound AI call handling. These are the numbers that callers can dial to reach your AI agents.

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Query Parameters

<ParamField query="status" type="string">
  Filter by number status. Options: `active`, `inactive`, `pending`.
</ParamField>

<ParamField query="country" type="string">
  Filter by country code (ISO 3166-1 alpha-2). Example: `US`, `GB`, `CA`.
</ParamField>

<ParamField query="page" type="integer" default={1}>
  Page number for pagination.
</ParamField>

<ParamField query="limit" type="integer" default={20}>
  Number of results per page (max: 100).
</ParamField>

## Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.talkos.io/api/inbound/bot-numbers?status=active&limit=10" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript Node.js
const response = await fetch('https://api.talkos.io/api/inbound/bot-numbers?status=active&limit=10', {
  method: 'GET',
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/inbound/bot-numbers',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    },
    params={
        'status': 'active',
        'limit': 10
    }
)

data = response.json()
```
</CodeGroup>

## Response Fields

<ResponseField name="data.numbers" type="array">
  List of phone numbers configured for inbound handling.
</ResponseField>

<ResponseField name="data.numbers[].id" type="string">
  Unique identifier for the phone number.
</ResponseField>

<ResponseField name="data.numbers[].phoneNumber" type="string">
  Phone number in E.164 format.
</ResponseField>

<ResponseField name="data.numbers[].friendlyName" type="string">
  Human-readable name for the number.
</ResponseField>

<ResponseField name="data.numbers[].capabilities" type="object">
  Supported features for this number (voice, sms, mms).
</ResponseField>

<ResponseField name="data.numbers[].status" type="string">
  Current status: `active`, `inactive`, or `pending`.
</ResponseField>

<ResponseField name="data.numbers[].webhookOverride" type="string">
  Number-specific webhook URL. If set, overrides tenant-level webhook.
</ResponseField>

<ResponseField name="data.numbers[].voiceIdOverride" type="string">
  Number-specific voice. If set, overrides tenant-level voice setting.
</ResponseField>

<ResponseField name="data.numbers[].stats" type="object">
  Call statistics for this specific number.
</ResponseField>

<Tip>
  Use `webhookOverride` to route different phone numbers to different backend handlers - useful for multi-department setups.
</Tip>
