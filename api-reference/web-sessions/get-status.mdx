---
title: "Get Session Status"
api: "GET https://api.talkos.io/api/web-sessions/{sessionId}/status"
description: "Get the current status of a web session"
---

Retrieve the current status and details of a specific web session.

## Path Parameters

<ParamField path="sessionId" type="string" required>
  The unique session identifier.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://api.talkos.io/api/web-sessions/ws-42e299e5-6bbf-4503-af0a-f8218524f8cf/status \
  -H "x-api-key: your-api-key"
```
</RequestExample>
