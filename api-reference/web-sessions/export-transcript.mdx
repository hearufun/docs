---
title: "Export Transcript"
api: "GET https://api.talkos.io/api/web-sessions/{sessionId}/transcript/export"
description: "Export the session transcript as a downloadable file"
---

Export the conversation transcript for a web session in various formats (JSON, text, or CSV).

## Path Parameters

<ParamField path="sessionId" type="string" required>
  The unique session identifier.
</ParamField>

## Query Parameters

<ParamField query="format" type="string" default="json">
  Export format. Supported values: `json`, `text`, `csv`.
</ParamField>

<RequestExample>
```bash cURL (JSON)
curl -X GET "https://api.talkos.io/api/web-sessions/ws-42e299e5-6bbf-4503-af0a-f8218524f8cf/transcript/export?format=json" \
  -H "x-api-key: your-api-key" \
  -o transcript.json
```

```bash cURL (Text)
curl -X GET "https://api.talkos.io/api/web-sessions/ws-42e299e5-6bbf-4503-af0a-f8218524f8cf/transcript/export?format=text" \
  -H "x-api-key: your-api-key" \
  -o transcript.txt
```

```bash cURL (CSV)
curl -X GET "https://api.talkos.io/api/web-sessions/ws-42e299e5-6bbf-4503-af0a-f8218524f8cf/transcript/export?format=csv" \
  -H "x-api-key: your-api-key" \
  -o transcript.csv
```
</RequestExample>

## Response Headers

The response includes headers for file download:

| Header | Value |
|--------|-------|
| `Content-Type` | `application/json`, `text/plain`, or `text/csv` |
| `Content-Disposition` | `attachment; filename="transcript-{sessionId}.{ext}"` |
