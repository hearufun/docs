---
title: "Get Session Statistics"
api: "GET https://api.talkos.io/api/web-sessions/stats"
description: "Get aggregate statistics for web sessions"
---

Retrieve aggregate statistics for web sessions including counts, durations, and usage metrics.

<RequestExample>
```bash cURL
curl -X GET https://api.talkos.io/api/web-sessions/stats \
  -H "x-api-key: your-api-key"
```
</RequestExample>
