---
title: "Search Sessions"
api: "GET https://api.talkos.io/api/web-sessions/search"
description: "Search and filter past web sessions with pagination"
---

Search and filter past web sessions with support for multiple criteria and pagination.

## Query Parameters

<ParamField query="status" type="string">
  Filter by session status. Values: `active`, `completed`, `paused`, `failed`.
</ParamField>

<ParamField query="agentType" type="string">
  Filter by agent type (e.g., "interviewer", "support", "sales").
</ParamField>

<ParamField query="language" type="string">
  Filter by language code (e.g., "en", "hi", "es").
</ParamField>

<ParamField query="endReason" type="string">
  Filter by session end reason.
</ParamField>

<ParamField query="dateFrom" type="string">
  Filter sessions created after this date (ISO 8601 format).
</ParamField>

<ParamField query="dateTo" type="string">
  Filter sessions created before this date (ISO 8601 format).
</ParamField>

<ParamField query="minDuration" type="integer">
  Minimum session duration in seconds.
</ParamField>

<ParamField query="maxDuration" type="integer">
  Maximum session duration in seconds.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number for pagination.
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Number of results per page.
</ParamField>

<ParamField query="sortBy" type="string" default="createdAt">
  Field to sort by.
</ParamField>

<ParamField query="sortOrder" type="string" default="desc">
  Sort order: `asc` or `desc`.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/web-sessions/search?status=completed&agentType=support&dateFrom=2026-02-01&limit=10&sortBy=createdAt&sortOrder=desc" \
  -H "x-api-key: your-api-key"
```
</RequestExample>
