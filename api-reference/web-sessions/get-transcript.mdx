---
title: "Get Session Transcript"
api: "GET https://api.talkos.io/api/web-sessions/{sessionId}/transcript"
description: "Get the full conversation transcript for a web session"
---

Retrieve the complete conversation transcript for a specific web session.

## Path Parameters

<ParamField path="sessionId" type="string" required>
  The unique session identifier.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://api.talkos.io/api/web-sessions/ws-42e299e5-6bbf-4503-af0a-f8218524f8cf/transcript \
  -H "x-api-key: your-api-key"
```
</RequestExample>
