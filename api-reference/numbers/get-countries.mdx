---
title: "Get Supported Countries"
api: "GET https://api.talkos.io/api/numbers/countries"
description: "Retrieve a list of countries where phone numbers are available for purchase"
---

Get a list of all supported countries where phone numbers can be purchased. Each country includes information about available number types and supported dialer providers.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="countries" type="array">
      Array of supported countries
      <Expandable title="country object">
        <ResponseField name="code" type="string">
          ISO 3166-1 alpha-2 country code
        </ResponseField>
        <ResponseField name="name" type="string">
          Full country name
        </ResponseField>
        <ResponseField name="dialCode" type="string">
          International dialing code (e.g., +1, +91)
        </ResponseField>
        <ResponseField name="numberTypes" type="array">
          Available number types in this country
        </ResponseField>
        <ResponseField name="providers" type="array">
          Supported dialer providers for this country
        </ResponseField>
        <ResponseField name="currency" type="string">
          Default billing currency for this country
        </ResponseField>
        <ResponseField name="requiresAddress" type="boolean">
          Whether address verification is required
        </ResponseField>
        <ResponseField name="requiresIdentity" type="boolean">
          Whether identity verification is required
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="count" type="integer">
      Total number of supported countries
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/numbers/countries" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript Node.js
const response = await fetch(
  'https://api.talkos.io/api/numbers/countries',
  {
    method: 'GET',
    headers: {
      'x-api-key': 'your-api-key',
      'x-tenant-id': 'your-tenant-id'
    }
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/numbers/countries',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    }
)

data = response.json()
```
