---
title: "Get Inventory Statistics"
api: "GET https://api.talkos.io/api/numbers/inventory/stats"
description: "Retrieve statistics and analytics for your phone number inventory"
---

Get comprehensive statistics about your phone number inventory including counts by status, type, country, and provider. Useful for monitoring inventory health and planning capacity.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Query Parameters

<ParamField query="includeUsage" type="boolean" default="false">
  Include detailed usage statistics for each category
</ParamField>

<ParamField query="includeHistory" type="boolean" default="false">
  Include historical trend data (last 30 days)
</ParamField>

<ParamField query="groupBy" type="string">
  Additional grouping for statistics
  - `country` - Group by country
  - `type` - Group by number type
  - `provider` - Group by dialer provider
  - `tags` - Group by tags
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/numbers/inventory/stats?includeUsage=true" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL (with history)
curl -X GET "https://api.talkos.io/api/numbers/inventory/stats?includeUsage=true&includeHistory=true" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript Node.js
const response = await fetch(
  'https://api.talkos.io/api/numbers/inventory/stats?includeUsage=true',
  {
    method: 'GET',
    headers: {
      'x-api-key': 'your-api-key',
      'x-tenant-id': 'your-tenant-id'
    }
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/numbers/inventory/stats',
    params={
        'includeUsage': True
    },
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    }
)

data = response.json()
```

## Use Cases

- **Capacity Planning**: Monitor `capacity.remaining` to know when to purchase more numbers
- **Cost Optimization**: Use `billing` data to identify underutilized numbers
- **Health Monitoring**: Check `alerts` for issues requiring attention
- **Trend Analysis**: Use `history` data to understand inventory changes over time
