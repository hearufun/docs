---
title: "Update Phone Number"
api: "PUT https://api.talkos.io/api/numbers/{identifier}"
description: "Update configuration and settings for a phone number"
---

Update the configuration, webhook URLs, friendly name, and other settings for an existing phone number.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Path Parameters

<ParamField path="identifier" type="string" required>
  The phone number ID or E.164 formatted number (e.g., `num_abc123xyz` or `+14155551234`)
</ParamField>

## Request Body

<ParamField body="friendlyName" type="string">
  Update the friendly name or label for the number
</ParamField>

<ParamField body="direction" type="string">
  Update call direction configuration
  - `outbound` - Outbound calls only
  - `inbound` - Inbound calls only
  - `both` - Both inbound and outbound calls
</ParamField>

<ParamField body="webhookUrl" type="string">
  Webhook URL for inbound call notifications
</ParamField>

<ParamField body="fallbackUrl" type="string">
  Fallback webhook URL if primary webhook fails
</ParamField>

<ParamField body="voicemailEnabled" type="boolean">
  Enable or disable voicemail for this number
</ParamField>

<ParamField body="recordingEnabled" type="boolean">
  Enable or disable automatic call recording
</ParamField>

<ParamField body="transcriptionEnabled" type="boolean">
  Enable or disable automatic transcription for recordings
</ParamField>

<ParamField body="tags" type="object">
  Custom key-value tags for organizing numbers. Replaces existing tags.
</ParamField>

<ParamField body="agentId" type="string">
  Associate an AI agent with this number for inbound calls
</ParamField>

<ParamField body="status" type="string">
  Update number status (limited to `active` or `suspended`)
</ParamField>
