---
title: "Release Phone Number"
api: "DELETE https://api.talkos.io/api/numbers/{identifier}"
description: "Release and cancel a phone number subscription"
---

Release a phone number back to the provider. This action cancels your subscription and the number will no longer be available for making or receiving calls. This action is irreversible.

<Warning>
  Releasing a number is permanent. The number will be returned to the provider's pool and may be reassigned to another customer. Ensure you have migrated any services using this number before releasing.
</Warning>

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Path Parameters

<ParamField path="identifier" type="string" required>
  The phone number ID or E.164 formatted number (e.g., `num_abc123xyz` or `+14155551234`)
</ParamField>

## Query Parameters

<ParamField query="force" type="boolean" default="false">
  Force release even if the number has active calls or pending webhooks. Use with caution.
</ParamField>

<ParamField query="reason" type="string">
  Optional reason for releasing the number (for audit purposes)
</ParamField>

## Notes

- **Prorated refunds**: If you release a number before the end of your billing cycle, you may receive a prorated refund for unused days.
- **Data retention**: Call logs, recordings, and transcripts associated with this number are retained according to your tenant's data retention policy.
- **Grace period**: Some providers offer a grace period where you can reclaim a released number. Contact support for assistance.
