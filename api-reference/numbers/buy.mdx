---
title: "Buy Phone Number"
api: "POST https://api.talkos.io/api/numbers/buy"
description: "Purchase a phone number from a supported dialer provider"
---

Purchase a new phone number from a supported dialer provider. The number will be provisioned and immediately available for use with your TalkOS tenant.

## Authentication

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Request Body

<ParamField body="dialerProvider" type="string" required>
  The dialer provider to purchase from
  - `plivo` - Plivo Communications
  - `exotel` - Exotel (India)
</ParamField>

<ParamField body="country" type="string" required>
  ISO 3166-1 alpha-2 country code (e.g., `US`, `IN`, `GB`)
</ParamField>

<ParamField body="type" type="string" required>
  Type of phone number to purchase
  - `local` - Local geographic number
  - `toll-free` - Toll-free number
  - `mobile` - Mobile number
</ParamField>

<ParamField body="areaCode" type="string">
  Preferred area code or prefix. If not available, a nearby area code may be assigned.
</ParamField>

<ParamField body="number" type="string">
  Specific phone number to purchase (from search results). If not provided, a random number matching criteria will be assigned.
</ParamField>

<ParamField body="direction" type="string" default="both">
  Call direction configuration for the number
  - `outbound` - Outbound calls only
  - `inbound` - Inbound calls only
  - `both` - Both inbound and outbound calls
</ParamField>

<ParamField body="friendlyName" type="string">
  A friendly name or label for the number (e.g., "Sales Hotline", "Support Line")
</ParamField>

<ParamField body="webhookUrl" type="string">
  Webhook URL for inbound call notifications
</ParamField>

<ParamField body="fallbackUrl" type="string">
  Fallback webhook URL if primary webhook fails
</ParamField>

<ParamField body="tags" type="object">
  Custom key-value tags for organizing numbers
</ParamField>

  Indicates if the request was successful

    </ResponseField>

```bash cURL
curl -X POST "https://api.talkos.io/api/numbers/buy" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id" \
  -H "Content-Type: application/json" \
  -d '{
    "dialerProvider": "plivo",
    "country": "US",
    "type": "local",
    "areaCode": "415",
    "direction": "both",
    "friendlyName": "Sales Hotline",
    "webhookUrl": "https://your-server.com/webhooks/inbound"
  }'
```
