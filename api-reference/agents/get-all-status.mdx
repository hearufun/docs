---
title: 'Get All Agent Status'
api: 'GET https://api.talkos.io/api/agents/status'
description: 'Retrieve the current status of all registered agents'
---

## Overview

Retrieves the current availability status and real-time metrics for all registered agents in your tenant. Use this endpoint to monitor agent availability, workload distribution, and team capacity.

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Query Parameters

<ParamField query="status" type="string">
  Filter by agent status: `available`, `busy`, `offline`, `break`
</ParamField>

<ParamField query="skill" type="string">
  Filter agents by a specific skill
</ParamField>

<ParamField query="language" type="string">
  Filter agents by supported language (ISO 639-1 code)
</ParamField>

<ParamField query="includeMetrics" type="boolean" default="false">
  Include today's performance metrics for each agent
</ParamField>

<ParamField query="includeCapacity" type="boolean" default="true">
  Include capacity information (current/max calls)
</ParamField>

<ParamField query="sortBy" type="string" default="name">
  Sort results by: `name`, `status`, `activeCalls`, `priority`, `lastActive`
</ParamField>

<ParamField query="sortOrder" type="string" default="asc">
  Sort direction: `asc` or `desc`
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Number of agents per page (max 100)
</ParamField>

<RequestExample>
```bash cURL - All Agents
curl -X GET "https://api.talkos.io/api/agents/status" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL - Available Only
curl -X GET "https://api.talkos.io/api/agents/status?status=available" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL - By Skill
curl -X GET "https://api.talkos.io/api/agents/status?skill=billing&status=available" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL - With Metrics
curl -X GET "https://api.talkos.io/api/agents/status?includeMetrics=true&sortBy=activeCalls&sortOrder=desc" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL - Spanish Speakers
curl -X GET "https://api.talkos.io/api/agents/status?language=es&status=available" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```
</RequestExample>

## Real-Time Dashboard Integration

This endpoint is designed for real-time dashboard updates. Recommended polling interval: 5-10 seconds.

For real-time updates without polling, use our WebSocket connection:

```javascript
const ws = new WebSocket('wss://api.talkos.io/realtime');
ws.send(JSON.stringify({
  type: 'subscribe',
  channel: 'agent-status',
  tenantId: 'your-tenant-id'
}));
```

## Error Codes

| Code | Description |
|------|-------------|
| `INVALID_STATUS_FILTER` | Status filter value not recognized |
| `INVALID_SORT_FIELD` | Sort field not recognized |
| `INVALID_SORT_ORDER` | Sort order must be asc or desc |
| `INVALID_PAGINATION` | Page or limit values invalid |
| `SKILL_NOT_FOUND` | Specified skill does not exist |
