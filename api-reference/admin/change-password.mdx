---
title: "Change Password"
api: "POST https://api.talkos.io/api/auth/admin/change-password"
description: "Change the current admin user's password"
---

Change the password for the currently authenticated admin user. Requires verification of the current password.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token from login. Format: `Bearer {token}`
</ParamField>

## Request Body

<ParamField body="currentPassword" type="string" required>
  The admin's current password for verification.
</ParamField>

<ParamField body="newPassword" type="string" required>
  The new password. Must be at least 6 characters.
</ParamField>

<ParamField body="confirmPassword" type="string" required>
  Confirmation of the new password. Must match newPassword.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the password change was successful.
</ResponseField>

<ResponseField name="message" type="string">
  Status message describing the result.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.talkos.io/api/auth/admin/change-password \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -H "Content-Type: application/json" \
  -d '{
    "currentPassword": "oldPassword123",
    "newPassword": "newSecurePassword456",
    "confirmPassword": "newSecurePassword456"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true,
  "message": "Password changed successfully"
}
```

```json 400
{
  "error": "Passwords do not match"
}
```

```json 400
{
  "error": "Password must be at least 6 characters"
}
```

```json 401
{
  "error": "Current password is incorrect"
}
```
</ResponseExample>

<Warning>
After changing your password, all existing sessions remain valid. For security, consider logging out from all devices after a password change.
</Warning>
