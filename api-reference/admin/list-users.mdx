---
title: "List Admin Users"
api: "GET https://api.talkos.io/api/auth/admin/users"
description: "List all admin users in the system"
---

Retrieve a list of all admin users. This endpoint is restricted to users with the "admin" role.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token from login. Format: `Bearer {token}`
</ParamField>

## Authorization

This endpoint requires the `admin` role. Regular users will receive a 403 Forbidden response.

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful.
</ResponseField>

<ResponseField name="count" type="number">
  Total number of admin users.
</ResponseField>

<ResponseField name="users" type="array">
  Array of admin user objects.
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Unique identifier for the admin user.
    </ResponseField>
    <ResponseField name="username" type="string">
      The admin's username.
    </ResponseField>
    <ResponseField name="email" type="string">
      The admin's email address.
    </ResponseField>
    <ResponseField name="role" type="string">
      The admin's role.
    </ResponseField>
    <ResponseField name="status" type="string">
      Account status (e.g., "active", "inactive").
    </ResponseField>
    <ResponseField name="permissions" type="object">
      The admin's permissions.
    </ResponseField>
    <ResponseField name="lastLogin" type="string">
      ISO 8601 timestamp of last login.
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp when account was created.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL

  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```
