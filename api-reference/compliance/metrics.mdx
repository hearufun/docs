---
title: "Get Compliance Metrics"
api: "GET https://api.talkos.io/api/compliance/metrics"
description: "Retrieve compliance metrics and statistics for your tenant"
---

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://api.talkos.io/api/compliance/metrics \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```
</RequestExample>
