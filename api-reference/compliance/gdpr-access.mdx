# GDPR Data Access

Submit a GDPR Article 15 data access request to retrieve all personal data associated with a user. Returns a comprehensive report of all data stored for the identified individual.

## Endpoint

```
POST /api/compliance/gdpr/access
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="email" type="string">
  Email address of the data subject. At least one identifier required.
</ParamField>

<ParamField body="phoneNumber" type="string">
  Phone number of the data subject in E.164 format. At least one identifier required.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was processed successfully
</ResponseField>

<ResponseField name="requestId" type="string">
  Unique identifier for this GDPR access request
</ResponseField>

<ResponseField name="status" type="string">
  Current status: `processing`, `completed`, `no_data_found`
</ResponseField>

<ResponseField name="dataSubject" type="object">
  Identifiers used to locate the data subject
</ResponseField>

<ResponseField name="data" type="object">
  All personal data found for the data subject
</ResponseField>

<ResponseField name="data.profile" type="object">
  User profile information
</ResponseField>

<ResponseField name="data.calls" type="array">
  Call records and metadata
</ResponseField>

<ResponseField name="data.transcripts" type="array">
  Call transcripts (if stored)
</ResponseField>

<ResponseField name="data.voicePrint" type="object">
  Voice biometrics enrollment data (if enrolled)
</ResponseField>

<ResponseField name="data.analytics" type="array">
  Analytics and behavioral data
</ResponseField>

<ResponseField name="generatedAt" type="string">
  ISO 8601 timestamp when the report was generated
</ResponseField>

<ResponseField name="expiresAt" type="string">
  ISO 8601 timestamp when the download link expires
</ResponseField>

<ResponseField name="downloadUrl" type="string">
  URL to download the full data export (valid for 7 days)
</ResponseField>

## Data Categories Included

| Category | Description |
|----------|-------------|
| Profile | Basic contact information and account details |
| Calls | Call history, metadata, and recordings |
| Transcripts | Call transcripts (redacted if PII redaction is enabled) |
| Voice Print | Voice biometrics enrollment and samples metadata |
| Analytics | Aggregated behavioral and sentiment data |

## Notes

<Info>
GDPR requires responding to data access requests within 30 days. This API provides immediate access to structured data. For large datasets, the response may return a `processing` status with a webhook notification when complete.
</Info>

<Warning>
Ensure you verify the identity of the person making the data access request before providing the data. This API does not verify data subject identity.
</Warning>

<Tip>
Set up a webhook to receive notifications when GDPR requests are completed. Configure webhooks in your tenant settings.
</Tip>
