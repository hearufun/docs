---
title: "Query Audit Log"
api: "GET https://api.talkos.io/api/compliance/audit"
description: "Query the audit log for compliance-related events and actions"
---

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Query Parameters

<ParamField query="startDate" type="string">
  Start date for the query range (ISO 8601 format)
  
  **Example:** `2026-02-01T00:00:00Z`
</ParamField>

<ParamField query="endDate" type="string">
  End date for the query range (ISO 8601 format)
  
  **Example:** `2026-02-18T23:59:59Z`
</ParamField>

<ParamField query="action" type="string">
  Filter by specific action type
  
  **Examples:** `consent_recorded`, `data_deleted`, `export_requested`, `access_granted`
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of records to return
  
  **Default:** 50  
  **Maximum:** 500
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/compliance/audit?startDate=2026-02-01T00:00:00Z&endDate=2026-02-18T23:59:59Z&action=consent_recorded&limit=100" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```
</RequestExample>
