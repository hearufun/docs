# Detect PII

Scan text for personally identifiable information (PII) and return the detected entities with their locations. Useful for compliance auditing and data protection workflows.

## Endpoint

```
POST /api/compliance/pii/detect
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="text" type="string" required>
  The text content to scan for PII (max 50,000 characters)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the detection was performed successfully
</ResponseField>

<ResponseField name="hasPii" type="boolean">
  Whether any PII was detected in the text
</ResponseField>

<ResponseField name="entities" type="array">
  Array of detected PII entities
</ResponseField>

<ResponseField name="entities[].type" type="string">
  Type of PII detected (e.g., `EMAIL`, `PHONE`, `SSN`, `CREDIT_CARD`)
</ResponseField>

<ResponseField name="entities[].value" type="string">
  The detected PII value
</ResponseField>

<ResponseField name="entities[].start" type="number">
  Starting character position in the text
</ResponseField>

<ResponseField name="entities[].end" type="number">
  Ending character position in the text
</ResponseField>

<ResponseField name="entities[].confidence" type="number">
  Confidence score from 0.0 to 1.0
</ResponseField>

<ResponseField name="summary" type="object">
  Summary of PII types and counts detected
</ResponseField>

## Supported PII Types

| Type | Description | Example |
|------|-------------|---------|
| `PERSON_NAME` | Full names | John Smith |
| `EMAIL` | Email addresses | john@example.com |
| `PHONE` | Phone numbers | 415-555-1234 |
| `SSN` | Social Security Numbers | 123-45-6789 |
| `CREDIT_CARD` | Credit card numbers | 4111-1111-1111-1111 |
| `ADDRESS` | Physical addresses | 123 Main St, City |
| `DATE_OF_BIRTH` | Birth dates | 01/15/1990 |
| `DRIVER_LICENSE` | Driver's license numbers | D1234567 |
| `PASSPORT` | Passport numbers | AB1234567 |
| `IP_ADDRESS` | IP addresses | 192.168.1.1 |
| `BANK_ACCOUNT` | Bank account numbers | 1234567890 |

## Notes

<Info>
Detection is performed locally and text is not stored. For automatic redaction, use the [Redact PII](/api-reference/compliance/redact-pii) endpoint.
</Info>

<Tip>
Use this endpoint to audit transcripts and logs before storage to ensure compliance with data protection regulations.
</Tip>
