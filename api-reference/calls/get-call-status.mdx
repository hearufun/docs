# Get Call Status

Retrieve real-time status, duration, and detailed information about a specific call using its unique identifier.

## Endpoint

```
GET /api/calls/:callUuid/status
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="callUuid" type="string" required>
  Unique identifier of the call (returned from Start Call endpoint)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="call" type="object">
  Call details object containing:

  <Expandable title="call properties">
    <ResponseField name="callUuid" type="string">
      Unique identifier for the call
    </ResponseField>

    <ResponseField name="status" type="string">
      Current call status:
      - `queued` - Call is waiting in the queue
      - `dialing` - Call is being placed
      - `ringing` - Phone is ringing
      - `in-progress` - Call is active
      - `completed` - Call ended normally
      - `failed` - Call failed to connect
      - `busy` - Recipient line was busy
      - `no-answer` - Call was not answered
      - `canceled` - Call was canceled before connecting
    </ResponseField>

    <ResponseField name="to" type="string">
      Destination phone number
    </ResponseField>

    <ResponseField name="from" type="string">
      Caller ID used for the call
    </ResponseField>

    <ResponseField name="direction" type="string">
      Call direction: `outbound` or `inbound`
    </ResponseField>

    <ResponseField name="startTime" type="string">
      ISO 8601 timestamp when the call started
    </ResponseField>

    <ResponseField name="answerTime" type="string">
      ISO 8601 timestamp when the call was answered (null if not answered)
    </ResponseField>

    <ResponseField name="endTime" type="string">
      ISO 8601 timestamp when the call ended (null if still active)
    </ResponseField>

    <ResponseField name="duration" type="number">
      Call duration in seconds (0 if not yet answered)
    </ResponseField>

    <ResponseField name="billableDuration" type="number">
      Billable duration in seconds (rounded up to nearest minute)
    </ResponseField>

    <ResponseField name="hangupCause" type="string">
      Reason for call termination:
      - `normal_clearing` - Normal hangup
      - `user_busy` - User was busy
      - `no_answer` - No answer
      - `call_rejected` - Call was rejected
      - `max_duration_reached` - Hit maximum duration limit
      - `api_hangup` - Ended via API
      - `error` - Technical error
    </ResponseField>

    <ResponseField name="hangupSource" type="string">
      Who ended the call: `caller`, `callee`, `system`, or `api`
    </ResponseField>

    <ResponseField name="recordingUrl" type="string">
      URL to download call recording (null if recording disabled or call not ended)
    </ResponseField>

    <ResponseField name="transcriptAvailable" type="boolean">
      Whether transcript is available for this call
    </ResponseField>

    <ResponseField name="analyticsAvailable" type="boolean">
      Whether AI analytics have been generated for this call
    </ResponseField>

    <ResponseField name="metadata" type="object">
      Custom metadata attached to the call
    </ResponseField>

    <ResponseField name="language" type="string">
      Language used for the call
    </ResponseField>

    <ResponseField name="dialerProvider" type="string">
      Telephony provider used: `plivo` or `exotel`
    </ResponseField>

    <ResponseField name="cost" type="object">
      Cost breakdown (only available after call ends):
      - `total` (number): Total cost in credits
      - `telephony` (number): Telephony cost
      - `ai` (number): AI processing cost
      - `currency` (string): Currency code
    </ResponseField>

    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp of call creation
    </ResponseField>

    <ResponseField name="updatedAt" type="string">
      ISO 8601 timestamp of last status update
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.talkos.io/api/calls/call_a1b2c3d4-e5f6-7890-abcd-ef1234567890/status" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript Node.js
const response = await fetch(
  'https://api.talkos.io/api/calls/call_a1b2c3d4-e5f6-7890-abcd-ef1234567890/status',
  {
    method: 'GET',
    headers: {
      'x-api-key': 'your-api-key',
      'x-tenant-id': 'your-tenant-id'
    }
  }
);

const data = await response.json();
console.log(data.call.status);
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/calls/call_a1b2c3d4-e5f6-7890-abcd-ef1234567890/status',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    }
)

data = response.json()
print(data['call']['status'])
```
</CodeGroup>

## Notes

- Call status is updated in real-time
- Recording URLs are only available after the call ends and processing completes
- Transcripts typically become available 30-60 seconds after call ends
- Analytics are generated asynchronously and may take up to 2 minutes after call ends
- Polling interval recommendation: every 5 seconds for active calls
