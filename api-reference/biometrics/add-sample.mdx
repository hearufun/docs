# Add Voice Sample

Add a voice sample from a call recording to strengthen an existing voice print profile. Multiple samples improve verification accuracy.

## Endpoint

```
POST /api/voice-biometrics/enroll/sample
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="callerId" type="string" required>
  The caller's phone number in E.164 format (e.g., +14155551234)
</ParamField>

<ParamField body="callUuid" type="string" required>
  Unique identifier of the call to extract the voice sample from
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the voice sample was added successfully
</ResponseField>

<ResponseField name="voicePrintId" type="string">
  Unique identifier for the voice print profile
</ResponseField>

<ResponseField name="callerId" type="string">
  The caller's phone number
</ResponseField>

<ResponseField name="sampleId" type="string">
  Unique identifier for the added voice sample
</ResponseField>

<ResponseField name="samplesCount" type="number">
  Total number of voice samples in the profile
</ResponseField>

<ResponseField name="samplesRequired" type="number">
  Number of additional samples needed (0 if enrollment is complete)
</ResponseField>

<ResponseField name="status" type="string">
  Current enrollment status: `pending`, `enrolled`, `active`
</ResponseField>

<ResponseField name="quality" type="object">
  Quality metrics for the extracted voice sample
</ResponseField>

## Notes

<Info>
The call must contain at least 10 seconds of clear speech from the caller for a valid voice sample extraction.
</Info>

<Tip>
For best results, collect voice samples from calls in different acoustic environments to improve verification accuracy.
</Tip>
