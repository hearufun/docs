# Get Voice Profile

Retrieve the voice biometrics profile for a specific caller, including enrollment status, sample count, and verification history.

## Endpoint

```
GET /api/voice-biometrics/:callerId
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="callerId" type="string" required>
  The caller's phone number in E.164 format (e.g., +14155551234). URL encode the `+` as `%2B`.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the profile was retrieved successfully
</ResponseField>

<ResponseField name="voicePrintId" type="string">
  Unique identifier for the voice print profile
</ResponseField>

<ResponseField name="callerId" type="string">
  The caller's phone number
</ResponseField>

<ResponseField name="callerName" type="string">
  The caller's registered name
</ResponseField>

<ResponseField name="status" type="string">
  Current enrollment status: `pending`, `enrolled`, `active`, `suspended`
</ResponseField>

<ResponseField name="samplesCount" type="number">
  Total number of voice samples in the profile
</ResponseField>

<ResponseField name="lastVerification" type="object">
  Details of the most recent verification attempt
</ResponseField>

<ResponseField name="verificationCount" type="number">
  Total number of verification attempts
</ResponseField>

<ResponseField name="successfulVerifications" type="number">
  Number of successful verifications
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp when the profile was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp when the profile was last updated
</ResponseField>

```bash cURL
curl -X GET "https://api.talkos.io/api/voice-biometrics/%2B14155551234" \
```

```javascript JavaScript
const callerId = encodeURIComponent('+14155551234');

    'x-api-key': 'your-api-key',
```

## Profile Status Values

| Status | Description |
|--------|-------------|
| `pending` | Profile created but needs more voice samples |
| `enrolled` | Sufficient samples collected, processing |
| `active` | Profile ready for verification |
| `suspended` | Profile temporarily disabled |

## Notes

<Info>
The `lastVerification` field is `null` if no verification attempts have been made yet.
</Info>

<Tip>
Use the verification success rate (`successfulVerifications / verificationCount`) to identify callers who may need to re-enroll due to voice changes.
</Tip>
