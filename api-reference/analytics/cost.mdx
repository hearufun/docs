## Overview

Get detailed cost analytics for your TalkOS usage. This endpoint provides comprehensive cost breakdowns by provider, call type, and time period to help you optimize spending and forecast budgets.

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Query Parameters

<ParamField query="startDate" type="string" required>
  Start date for the query range in ISO 8601 format
</ParamField>

<ParamField query="endDate" type="string" required>
  End date for the query range in ISO 8601 format
</ParamField>

<ParamField query="groupBy" type="string" default="day">
  Group results by time period. Options: `hour`, `day`, `week`, `month`
</ParamField>

<ParamField query="direction" type="string">
  Filter by call direction. Options: `inbound`, `outbound`
</ParamField>

<ParamField query="currency" type="string" default="USD">
  Currency for cost values. Options: `USD`, `EUR`, `GBP`
</ParamField>

const data = await response.json();

## Cost Components

<CardGroup cols={2}>
  <Card title="Telephony" icon="phone">
    Per-minute charges for call connectivity via carriers. Rates vary by destination and call direction.
  </Card>
  <Card title="LLM" icon="brain">
    Token-based charges for AI language model processing. Includes both input and output tokens.
  </Card>
  <Card title="STT (Speech-to-Text)" icon="microphone">
    Per-minute charges for real-time speech recognition and transcription.
  </Card>
  <Card title="TTS (Text-to-Speech)" icon="volume-up">
    Per-character charges for voice synthesis. Premium voices may have higher rates.
  </Card>
</CardGroup>

## Cost Optimization Tips

<AccordionGroup>
  <Accordion title="Optimize LLM Usage">
    LLM costs typically account for 25-35% of total cost. Reduce token usage by:
    - Using concise system prompts
    - Implementing context windowing
    - Caching common responses
  </Accordion>
  <Accordion title="Reduce Call Duration">
    Telephony is billed per minute. Optimize agents to:
    - Resolve issues faster
    - Avoid unnecessary confirmations
    - Use efficient conversation flows
  </Accordion>
  <Accordion title="Monitor Cost Spikes">
    Set up alerts for daily cost thresholds. Sudden spikes may indicate:
    - Traffic anomalies
    - Inefficient agent behavior
    - Configuration issues
  </Accordion>
  <Accordion title="Right-size TTS">
    TTS costs scale with output length. Reduce costs by:
    - Using shorter responses where appropriate
    - Avoiding repeated information
    - Using standard voices for non-critical calls
  </Accordion>
</AccordionGroup>

## Budget Alerts

Configure budget alerts in the TalkOS dashboard to receive notifications when:
- Daily spend exceeds a threshold
- Monthly projected spend exceeds budget
- Cost per call increases significantly

<Note>
  Cost data is updated every 15 minutes. Real-time cost tracking is available via the [WebSocket API](/api-reference/realtime/websocket) for Enterprise tier customers.
</Note>

## Rate Limits

- **Standard tier**: 30 requests per minute
- **Enterprise tier**: 120 requests per minute
