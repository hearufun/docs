## Overview

Retrieve historical call analytics with powerful filtering and pagination. Use this endpoint to analyze call patterns, export call data, and generate reports.

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Query Parameters

<ParamField query="startDate" type="string" required>
  Start date for the query range in ISO 8601 format (e.g., `2026-02-01T00:00:00Z`)
</ParamField>

<ParamField query="endDate" type="string" required>
  End date for the query range in ISO 8601 format (e.g., `2026-02-18T23:59:59Z`)
</ParamField>

<ParamField query="status" type="string">
  Filter by call status. Options: `completed`, `failed`, `abandoned`, `transferred`, `in-progress`
</ParamField>

<ParamField query="direction" type="string">
  Filter by call direction. Options: `inbound`, `outbound`
</ParamField>

<ParamField query="phoneNumber" type="string">
  Filter by phone number (caller or callee)
</ParamField>

<ParamField query="minDuration" type="integer">
  Minimum call duration in seconds
</ParamField>

<ParamField query="maxDuration" type="integer">
  Maximum call duration in seconds
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Number of records to return per page (max: 500)
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of records to skip for pagination
</ParamField>

<ParamField query="sortBy" type="string" default="startTime">
  Field to sort by. Options: `startTime`, `duration`, `status`
</ParamField>

<ParamField query="sortOrder" type="string" default="desc">
  Sort order. Options: `asc`, `desc`
</ParamField>

## Pagination

Use `limit` and `offset` parameters for pagination through large result sets:

```bash
# First page
curl "https://api.talkos.io/api/analytics/calls?startDate=2026-02-01T00:00:00Z&endDate=2026-02-18T23:59:59Z&limit=50&offset=0"

# Second page
curl "https://api.talkos.io/api/analytics/calls?startDate=2026-02-01T00:00:00Z&endDate=2026-02-18T23:59:59Z&limit=50&offset=50"
```

<Warning>
  The maximum date range is 90 days. For longer ranges, make multiple requests with different date windows.
</Warning>

## Rate Limits

- **Standard tier**: 30 requests per minute
- **Enterprise tier**: 120 requests per minute
