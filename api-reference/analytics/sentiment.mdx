---
title: "Get Aggregate Sentiment"
api: "GET https://api.talkos.io/api/analytics/sentiment"
description: "Retrieve aggregate sentiment analysis across all calls with positive, neutral, and negative breakdowns"
---

## Overview

Get aggregate sentiment analytics for your calls over a specified time period. This endpoint provides overall sentiment distribution, average sentiment scores, and trend data to help you understand customer satisfaction patterns.

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your tenant identifier
</ParamField>

## Query Parameters

<ParamField query="startDate" type="string" required>
  Start date for the query range in ISO 8601 format
</ParamField>

<ParamField query="endDate" type="string" required>
  End date for the query range in ISO 8601 format
</ParamField>

<ParamField query="direction" type="string">
  Filter by call direction. Options: `inbound`, `outbound`
</ParamField>

<ParamField query="groupBy" type="string" default="day">
  Group results by time period. Options: `hour`, `day`, `week`, `month`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="summary" type="object">
      <Expandable title="properties">
        <ResponseField name="totalCalls" type="integer">

        </ResponseField>
        <ResponseField name="positiveCount" type="integer">
          Number of calls with positive sentiment
        </ResponseField>
        <ResponseField name="neutralCount" type="integer">
          Number of calls with neutral sentiment
        </ResponseField>
        <ResponseField name="negativeCount" type="integer">

        </ResponseField>
        <ResponseField name="positivePercent" type="number">

        <ResponseField name="negativePercent" type="number">
          Percentage of calls with negative sentiment
        </ResponseField>
        <ResponseField name="avgSentimentScore" type="number">
          Average sentiment score (-1 to 1, where 1 is most positive)

      </Expandable>
    </ResponseField>

      Time-series sentiment data grouped by the specified period
      <Expandable title="trend object">
        <ResponseField name="period" type="string">
          Time period (date or date range)

        <ResponseField name="positiveCount" type="integer">
          Positive sentiment count for this period
        </ResponseField>
        <ResponseField name="neutralCount" type="integer">
          Neutral sentiment count for this period
        </ResponseField>
        <ResponseField name="negativeCount" type="integer">
          Negative sentiment count for this period

        <ResponseField name="avgScore" type="number">
          Average sentiment score for this period

    </ResponseField>

    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/analytics/sentiment?startDate=2026-02-01T00:00:00Z&endDate=2026-02-18T23:59:59Z&groupBy=day" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript JavaScript
const params = new URLSearchParams({
  startDate: '2026-02-01T00:00:00Z',
  endDate: '2026-02-18T23:59:59Z',
  groupBy: 'day'
});

const response = await fetch(`https://api.talkos.io/api/analytics/sentiment?${params}`, {
  method: 'GET',
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/analytics/sentiment',
    params={
        'startDate': '2026-02-01T00:00:00Z',
        'endDate': '2026-02-18T23:59:59Z',
        'groupBy': 'day'
    },
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    }
)

data = response.json()
```

## Understanding Sentiment Scores

<CardGroup cols={3}>
  <Card title="Positive" icon="face-smile">
    Score: 0.3 to 1.0  
    Customer expressed satisfaction, gratitude, or positive emotion
  </Card>
  <Card title="Neutral" icon="face-meh">
    Score: -0.3 to 0.3  
    Factual conversation without strong emotional indicators
  </Card>
  <Card title="Negative" icon="face-frown">
    Score: -1.0 to -0.3  
    Customer expressed frustration, anger, or dissatisfaction
  </Card>
</CardGroup>

## Best Practices

<AccordionGroup>
  <Accordion title="Monitor Sentiment Trends">
    Track daily sentiment trends to identify systemic issues before they escalate. A sudden drop in average sentiment score often indicates a service problem.
  </Accordion>
  <Accordion title="Act on Negative Reasons">
    Use the `topNegativeReasons` data to prioritize process improvements. Focus on the highest-impact issues first.
  </Accordion>
  <Accordion title="Compare Time Periods">
    Make multiple requests with different date ranges to compare sentiment before and after process changes.
  </Accordion>
</AccordionGroup>

## Rate Limits

- **Standard tier**: 30 requests per minute
- **Enterprise tier**: 120 requests per minute
