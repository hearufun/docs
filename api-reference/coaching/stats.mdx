# Get Coaching Stats

Retrieve comprehensive coaching statistics. When an agent ID is provided, returns stats specific to that agent. Otherwise, returns organization-wide coaching metrics.

## Endpoint

```
GET /api/coaching/stats/:agentId?
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="agentId" type="string">
  Optional. Agent ID to filter statistics for a specific agent. If omitted, returns organization-wide stats.
</ParamField>

## Query Parameters

<ParamField query="startDate" type="string">
  Filter stats from this date (ISO 8601 format). Defaults to 30 days ago.
</ParamField>

<ParamField query="endDate" type="string">
  Filter stats until this date (ISO 8601 format). Defaults to current date.
</ParamField>

<ParamField query="groupBy" type="string">
  Group statistics by: `day`, `week`, or `month`. Defaults to `day`.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="agentId" type="string">
  The agent ID if filtering by specific agent, otherwise null
</ResponseField>

<ResponseField name="period" type="object">
  The time period for the statistics

  <Expandable title="period properties">
    <ResponseField name="startDate" type="string">
      Start date of the statistics period
    </ResponseField>

    <ResponseField name="endDate" type="string">
      End date of the statistics period
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="overview" type="object">
  High-level coaching statistics

  <Expandable title="overview properties">
    <ResponseField name="totalSessions" type="number">
      Total number of coaching sessions
    </ResponseField>

    <ResponseField name="totalDuration" type="number">
      Total coaching duration in seconds
    </ResponseField>

    <ResponseField name="avgSessionDuration" type="number">
      Average session duration in seconds
    </ResponseField>

    <ResponseField name="totalMessages" type="number">
      Total messages sent across all sessions
    </ResponseField>

    <ResponseField name="avgMessagesPerSession" type="number">
      Average messages per coaching session
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="messageBreakdown" type="object">
  Statistics broken down by message type

  <Expandable title="messageBreakdown properties">
    <ResponseField name="suggestions" type="object">
      Stats for suggestion messages (count, percentage)
    </ResponseField>

    <ResponseField name="alerts" type="object">
      Stats for alert messages (count, percentage)
    </ResponseField>

    <ResponseField name="scripts" type="object">
      Stats for script messages (count, percentage)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="performance" type="object">
  Performance metrics

  <Expandable title="performance properties">
    <ResponseField name="avgResponseTime" type="number">
      Average time for agents to respond to coaching messages (seconds)
    </ResponseField>

    <ResponseField name="acknowledgmentRate" type="number">
      Percentage of messages acknowledged by agents
    </ResponseField>

    <ResponseField name="deliveryRate" type="number">
      Percentage of messages successfully delivered
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timeline" type="array">
  Statistics grouped by the specified time period

  <Expandable title="timeline item properties">
    <ResponseField name="date" type="string">
      Date for this data point
    </ResponseField>

    <ResponseField name="sessions" type="number">
      Number of sessions in this period
    </ResponseField>

    <ResponseField name="messages" type="number">
      Number of messages in this period
    </ResponseField>

    <ResponseField name="avgDuration" type="number">
      Average session duration in this period
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="topSupervisors" type="array">
  List of most active supervisors (only in org-wide stats)

  <Expandable title="supervisor properties">
    <ResponseField name="supervisorId" type="string">
      Supervisor identifier
    </ResponseField>

    <ResponseField name="sessionsCount" type="number">
      Number of coaching sessions conducted
    </ResponseField>

    <ResponseField name="totalMessages" type="number">
      Total messages sent
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL (Organization-wide)
curl -X GET "https://api.talkos.io/api/coaching/stats?startDate=2026-02-01&endDate=2026-02-18" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```bash cURL (Specific Agent)
curl -X GET "https://api.talkos.io/api/coaching/stats/agent-042?groupBy=week" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```


```python Python

    }
)
```

## Error Codes

| Code | Description |
|------|-------------|
| `AGENT_NOT_FOUND` | The specified agent ID does not exist |
| `INVALID_DATE_RANGE` | Start date must be before end date |
| `INVALID_GROUP_BY` | groupBy must be: day, week, or month |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
