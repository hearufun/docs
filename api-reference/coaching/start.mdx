# Start Coaching Session

Initiate a real-time coaching session that allows supervisors to provide live guidance, suggestions, and alerts to agents during active calls without the customer hearing.

## Endpoint

```
POST /api/coaching/start
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="callUuid" type="string" required>
  Unique identifier of the active call to coach
</ParamField>

<ParamField body="supervisorId" type="string" required>
  Unique identifier of the supervisor initiating the coaching session
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the coaching session was started successfully
</ResponseField>

<ResponseField name="coachingId" type="string">
  Unique identifier for the coaching session
</ResponseField>

<ResponseField name="callUuid" type="string">
  The call UUID associated with this coaching session
</ResponseField>

<ResponseField name="supervisorId" type="string">
  ID of the supervisor conducting the coaching
</ResponseField>

<ResponseField name="agentId" type="string">
  ID of the agent being coached
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the coaching session: `active`
</ResponseField>

<ResponseField name="startedAt" type="string">
  ISO 8601 timestamp when the coaching session started
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `CALL_NOT_FOUND` | The specified call UUID does not exist |
| `CALL_NOT_ACTIVE` | The call is not currently in progress |
| `COACHING_EXISTS` | A coaching session already exists for this call |
| `SUPERVISOR_NOT_FOUND` | The supervisor ID is invalid |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
