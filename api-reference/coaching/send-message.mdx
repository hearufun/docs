# Send Coaching Message

Send real-time coaching messages to agents during active calls. Messages are delivered instantly to the agent's interface without the customer hearing or seeing them.

## Endpoint

```
POST /api/coaching/:coachingId/message
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="coachingId" type="string" required>
  Unique identifier of the coaching session
</ParamField>

## Request Body

<ParamField body="message" type="string" required>
  The coaching message content to send to the agent
</ParamField>

<ParamField body="type" type="string" required>
  Type of coaching message:
  - `suggestion` - A helpful suggestion or tip for the agent
  - `alert` - An urgent notification requiring immediate attention
  - `script` - A pre-written script or response for the agent to use
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the message was sent successfully
</ResponseField>

<ResponseField name="messageId" type="string">
  Unique identifier for the sent message
</ResponseField>

<ResponseField name="coachingId" type="string">
  The coaching session ID
</ResponseField>

<ResponseField name="message" type="string">
  The message content that was sent
</ResponseField>

<ResponseField name="type" type="string">
  The type of message sent
</ResponseField>

<ResponseField name="sentAt" type="string">
  ISO 8601 timestamp when the message was sent
</ResponseField>

<ResponseField name="delivered" type="boolean">
  Whether the message was delivered to the agent's interface
</ResponseField>

## Message Types

| Type | Description | Use Case |
|------|-------------|----------|
| `suggestion` | Non-urgent guidance | Tips, recommendations, upsell opportunities |
| `alert` | Urgent notification | Compliance warnings, sentiment drops, escalation triggers |
| `script` | Word-for-word text | De-escalation scripts, legal disclaimers, promotional offers |

## Error Codes

| Code | Description |
|------|-------------|
| `SESSION_NOT_FOUND` | The coaching session does not exist |
| `SESSION_NOT_ACTIVE` | The coaching session has ended |
| `INVALID_MESSAGE_TYPE` | The message type must be: suggestion, alert, or script |
| `MESSAGE_REQUIRED` | Message content cannot be empty |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
