# Get Coaching Session

Retrieve detailed information about a specific coaching session, including all messages sent, participants, and session metrics.

## Endpoint

```
GET /api/coaching/:coachingId
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="coachingId" type="string" required>
  Unique identifier of the coaching session
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="session" type="object">
  Coaching session details

  <Expandable title="session properties">
    <ResponseField name="coachingId" type="string">
      Unique identifier for the coaching session
    </ResponseField>

    <ResponseField name="callUuid" type="string">
      The call UUID associated with this coaching session
    </ResponseField>

    <ResponseField name="supervisorId" type="string">
      ID of the supervisor conducting the coaching
    </ResponseField>

    <ResponseField name="agentId" type="string">
      ID of the agent being coached
    </ResponseField>

    <ResponseField name="status" type="string">
      Current status: `active` or `ended`
    </ResponseField>

    <ResponseField name="startedAt" type="string">
      ISO 8601 timestamp when the session started
    </ResponseField>

    <ResponseField name="endedAt" type="string">
      ISO 8601 timestamp when the session ended (null if active)
    </ResponseField>

    <ResponseField name="duration" type="number">
      Session duration in seconds (calculated in real-time if active)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="messages" type="array">
  Array of messages sent during the session

  <Expandable title="message properties">
    <ResponseField name="messageId" type="string">
      Unique identifier for the message
    </ResponseField>

    <ResponseField name="message" type="string">
      The message content
    </ResponseField>

    <ResponseField name="type" type="string">
      Message type: `suggestion`, `alert`, or `script`
    </ResponseField>

    <ResponseField name="sentAt" type="string">
      ISO 8601 timestamp when the message was sent
    </ResponseField>

    <ResponseField name="delivered" type="boolean">
      Whether the message was delivered to the agent
    </ResponseField>

    <ResponseField name="acknowledged" type="boolean">
      Whether the agent acknowledged the message
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="metrics" type="object">
  Session performance metrics

  <Expandable title="metrics properties">
    <ResponseField name="totalMessages" type="number">
      Total number of messages sent
    </ResponseField>

    <ResponseField name="byType" type="object">
      Message count breakdown by type
    </ResponseField>

    <ResponseField name="avgResponseTime" type="number">
      Average time in seconds for agent to acknowledge messages
    </ResponseField>

    <ResponseField name="deliveryRate" type="number">
      Percentage of messages successfully delivered
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.talkos.io/api/coaching/coach-789xyz \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript JavaScript
const response = await fetch('https://api.talkos.io/api/coaching/coach-789xyz', {
  method: 'GET',
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/coaching/coach-789xyz',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    }
)

data = response.json()
```

| Code | Description |
|------|-------------|
| `SESSION_NOT_FOUND` | The coaching session does not exist |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
