# Tag Analytics

Retrieve comprehensive analytics about tag usage across your calls, including most common tags, trends, and distribution.

## Endpoint

```
GET /api/features/tags/analytics/summary
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Response

<ResponseField name="totalTagsApplied" type="number">
  Total number of tag applications across all calls
</ResponseField>

<ResponseField name="uniqueTags" type="number">
  Number of unique tags in use
</ResponseField>

<ResponseField name="callsWithTags" type="number">
  Number of calls that have at least one tag
</ResponseField>

<ResponseField name="averageTagsPerCall" type="number">
  Average number of tags per call
</ResponseField>

<ResponseField name="topTags" type="array">
  Most frequently used tags
</ResponseField>

<ResponseField name="topTags[].tag" type="string">
  Tag name
</ResponseField>

<ResponseField name="topTags[].count" type="number">
  Number of times this tag has been applied
</ResponseField>

<ResponseField name="topTags[].percentage" type="number">
  Percentage of tagged calls that have this tag
</ResponseField>

<ResponseField name="tagsByCategory" type="object">
  Tag distribution by category
</ResponseField>

<ResponseField name="tagsByCategory.system" type="number">
  Count of system tag applications
</ResponseField>

<ResponseField name="tagsByCategory.custom" type="number">
  Count of custom tag applications
</ResponseField>

<ResponseField name="tagsByCategory.ai-generated" type="number">
  Count of AI-generated tag applications
</ResponseField>

<ResponseField name="tagsBySource" type="object">
  Tag distribution by source
</ResponseField>

<ResponseField name="tagsBySource.auto" type="number">
  Tags applied automatically
</ResponseField>

<ResponseField name="tagsBySource.manual" type="number">
  Tags applied manually by agents
</ResponseField>

<ResponseField name="tagsBySource.api" type="number">
  Tags applied via API
</ResponseField>

<ResponseField name="recentTrends" type="array">
  Recently trending tags
</ResponseField>

<ResponseField name="recentTrends[].tag" type="string">
  Tag name
</ResponseField>

<ResponseField name="recentTrends[].trend" type="string">
  Trend direction: `up`, `down`, or `stable`
</ResponseField>

<ResponseField name="recentTrends[].change" type="number">
  Percentage change from previous period
</ResponseField>

## Use Cases

- **Identify common issues**: Track which problems customers face most frequently
- **Monitor trends**: Detect increases in complaints or cancellation requests
- **Measure tagging adoption**: Track manual vs. automatic tagging ratios
- **Optimize auto-tagging**: Identify opportunities to add new auto-tag rules
- **Quality insights**: Analyze positive feedback trends
