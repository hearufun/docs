# Register Tag

Register a new custom tag that can be used for manual or automatic tagging of calls.

## Endpoint

```
POST /api/features/tags/register
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  Unique name for the tag (lowercase, hyphen-separated recommended)
</ParamField>

<ParamField body="displayName" type="string">
  Human-readable display name
</ParamField>

<ParamField body="description" type="string">
  Description of when this tag should be used
</ParamField>

<ParamField body="category" type="string">
  Tag category: `custom` (default), `department`, `outcome`, `priority`
</ParamField>

<ParamField body="color" type="string">
  Hex color code for visual identification (e.g., `#4CAF50`)
</ParamField>

<ParamField body="autoTagRule" type="object">
  Optional rule for automatic tagging
</ParamField>

<ParamField body="autoTagRule.keywords" type="array">
  Keywords that trigger this tag
</ParamField>

<ParamField body="autoTagRule.sentiment" type="object">
  Sentiment conditions (e.g., `{ "below": -0.5 }` or `{ "above": 0.7 }`)
</ParamField>

<ParamField body="autoTagRule.duration" type="object">
  Duration conditions in seconds (e.g., `{ "above": 600 }`)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the tag was registered successfully
</ResponseField>

<ResponseField name="tag" type="object">
  The registered tag details
</ResponseField>

<ResponseField name="tag.id" type="string">
  Unique identifier for the tag
</ResponseField>

<ResponseField name="tag.name" type="string">
  Tag name
</ResponseField>

<ResponseField name="tag.displayName" type="string">
  Display name
</ResponseField>

<ResponseField name="tag.category" type="string">
  Tag category
</ResponseField>

<ResponseField name="tag.color" type="string">
  Hex color code
</ResponseField>

<ResponseField name="tag.createdAt" type="string">
  ISO 8601 timestamp when the tag was created
</ResponseField>

## Tag Categories

| Category | Description | Use Case |
|----------|-------------|----------|
| `custom` | General purpose custom tags | Flexible categorization |
| `department` | Department or team tags | Routing analysis |
| `outcome` | Call outcome tags | Resolution tracking |
| `priority` | Priority level tags | Lead scoring, urgency |
