# Search Calls by Tags

Search for calls that have specific tags applied. Supports matching any or all specified tags.

## Endpoint

```
POST /api/features/tags/search/calls
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="tags" type="array" required>
  Array of tag names to search for
</ParamField>

<ParamField body="matchAll" type="boolean">
  If `true`, calls must have ALL specified tags. If `false` (default), calls matching ANY tag are returned.
</ParamField>

## Response

<ResponseField name="results" type="array">
  Array of matching calls
</ResponseField>

<ResponseField name="results[].callUuid" type="string">
  Unique identifier of the call
</ResponseField>

<ResponseField name="results[].tags" type="array">
  All tags on this call
</ResponseField>

<ResponseField name="results[].matchedTags" type="array">
  Tags that matched the search criteria
</ResponseField>

<ResponseField name="results[].createdAt" type="string">
  ISO 8601 timestamp when the call was created
</ResponseField>

<ResponseField name="results[].duration" type="number">
  Call duration in seconds
</ResponseField>

<ResponseField name="results[].status" type="string">
  Call status
</ResponseField>

<ResponseField name="count" type="number">
  Total number of matching calls
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.talkos.io/api/features/tags/search/calls \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id" \
  -d '{
    "tags": ["complaint", "escalated"],
    "matchAll": true
  }'
```

const response = await fetch('https://api.talkos.io/api/features/tags/search/calls', {

## Search Modes

| Mode | `matchAll` | Description |
|------|------------|-------------|
| Match Any | `false` | Returns calls with at least one of the specified tags |
| Match All | `true` | Returns only calls with ALL specified tags |
