# Configure Tagging

Configure the call tagging system for your tenant, including auto-tagging rules, custom tags, and tagging permissions.

## Endpoint

```
POST /api/features/tags/configure
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="autoTagEnabled" type="boolean">
  Enable automatic tagging based on keywords, sentiment, and context (default: true)
</ParamField>

<ParamField body="customTags" type="array">
  Array of custom tag definitions
</ParamField>

<ParamField body="customTags[].name" type="string">
  Tag name
</ParamField>

<ParamField body="customTags[].category" type="string">
  Tag category: `custom`, `system`, or `ai-generated`
</ParamField>

<ParamField body="customTags[].color" type="string">
  Hex color code for the tag (e.g., `#FF4444`)
</ParamField>

<ParamField body="additionalRules" type="array">
  Additional auto-tagging rules
</ParamField>

<ParamField body="additionalRules[].tag" type="string">
  Tag to apply when rule triggers
</ParamField>

<ParamField body="additionalRules[].triggers" type="object">
  Trigger conditions for the rule
</ParamField>

<ParamField body="maxTagsPerCall" type="number">
  Maximum number of tags allowed per call (default: 20)
</ParamField>

<ParamField body="allowAgentTagging" type="boolean">
  Allow agents to manually add/remove tags (default: true)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether configuration was successful
</ResponseField>

<ResponseField name="config" type="object">
  The complete tagging configuration
</ResponseField>

<ResponseField name="config.tenantId" type="string">
  Your tenant identifier
</ResponseField>

<ResponseField name="config.autoTagEnabled" type="boolean">
  Whether auto-tagging is enabled
</ResponseField>

<ResponseField name="config.customTags" type="array">
  Array of custom tags
</ResponseField>

<ResponseField name="config.autoTagRules" type="array">
  All active auto-tag rules (built-in + custom)
</ResponseField>

<ResponseField name="config.maxTagsPerCall" type="number">
  Maximum tags per call
</ResponseField>

<ResponseField name="config.allowAgentTagging" type="boolean">
  Whether agent tagging is allowed
</ResponseField>

<ResponseField name="config.updatedAt" type="number">
  Timestamp when config was last updated
</ResponseField>

## Built-in Auto-Tag Rules

TalkOS includes several built-in auto-tag rules that analyze call content:

| Tag | Trigger |
|-----|---------|
| `complaint` | Negative keywords + sentiment below -0.5 |
| `billing` | Billing-related keywords |
| `technical-issue` | Technical support keywords |
| `escalated` | Escalation request keywords |
| `positive-feedback` | Positive keywords + sentiment above 0.7 |
| `cancellation` | Cancellation-related keywords |
| `new-customer` | New customer keywords |
| `urgent` | Urgency keywords |
| `long-call` | Call duration > 10 minutes |
| `short-call` | Call duration < 30 seconds |
