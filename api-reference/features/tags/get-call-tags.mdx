# Get Call Tags

Retrieve all tags associated with a specific call, including the tagging history showing who added or removed tags and when.

## Endpoint

```
GET /api/features/tags/:callUuid
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="callUuid" type="string" required>
  Unique identifier of the call
</ParamField>

## Response

<ResponseField name="callUuid" type="string">
  The call UUID
</ResponseField>

<ResponseField name="tags" type="array">
  Array of current tags on the call
</ResponseField>

<ResponseField name="history" type="array">
  Complete history of tag changes
</ResponseField>

<ResponseField name="history[].action" type="string">
  Action type: `added` or `removed`
</ResponseField>

<ResponseField name="history[].tag" type="string">
  The tag that was added or removed
</ResponseField>

<ResponseField name="history[].by" type="string">
  Who performed the action
</ResponseField>

<ResponseField name="history[].at" type="string">
  ISO 8601 timestamp of the action
</ResponseField>

<ResponseField name="history[].source" type="string">
  Source of the tag: `manual`, `auto`, or `api`
</ResponseField>

```bash cURL

```
