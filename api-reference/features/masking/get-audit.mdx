---
title: 'Get Masking Audit Log'
api: 'GET https://api.talkos.io/api/features/masking/{callUuid}/audit'
description: 'Get the audit log for masking activities on a specific call'
---

Retrieve the complete audit trail of all masking-related activities for a specific call, including unmask requests, approvals, and denials.

## Path Parameters

<ParamField path="callUuid" type="string" required>
  The unique identifier of the call.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful.
</ResponseField>

<ResponseField name="callUuid" type="string">
  The call's unique identifier.
</ResponseField>

<ResponseField name="entries" type="array">
  Array of audit log entries.
  <Expandable title="entry properties">
    <ResponseField name="timestamp" type="string">
      ISO 8601 timestamp of the event.
    </ResponseField>
    <ResponseField name="action" type="string">
      The action type: `masked`, `unmask_requested`, `unmask_approved`, `unmask_denied`, `viewed`.
    </ResponseField>
    <ResponseField name="actor" type="string">
      The identifier of the user who performed the action.
    </ResponseField>
    <ResponseField name="actorRole" type="string">
      The role of the actor: `system`, `agent`, `supervisor`, `admin`.
    </ResponseField>
    <ResponseField name="details" type="object">
      Additional details about the action.
    </ResponseField>
    <ResponseField name="ipAddress" type="string">
      The IP address from which the action was performed.
    </ResponseField>
  </Expandable>
</ResponseField>
