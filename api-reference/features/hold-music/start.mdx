---
title: 'Start Hold Music'
api: 'POST https://api.talkos.io/api/features/hold-music/{callUuid}/start'
description: 'Start playing hold music on a call'
---

Begin playing hold music on an active call. This is typically used when placing a caller on hold during a transfer or while waiting for an agent.

## Path Parameters

<ParamField path="callUuid" type="string" required>
  The unique identifier of the call.
</ParamField>

## Request Body

<ParamField body="trackId" type="string">
  Specific track to play. If not provided, uses the default track.
</ParamField>

<ParamField body="volume" type="number">
  Override the default volume (0-100).
</ParamField>

<ParamField body="loop" type="boolean" default={true}>
  Whether to loop the track continuously.
</ParamField>

<ParamField body="announcePosition" type="boolean" default={true}>
  Whether to announce queue position at configured intervals.
</ParamField>

<ParamField body="estimatedWaitTime" type="number">
  Estimated wait time in seconds to announce to the caller.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether hold music was started successfully.
</ResponseField>

<ResponseField name="holdId" type="string">
  Unique identifier for this hold session.
</ResponseField>

<ResponseField name="callUuid" type="string">
  The call's unique identifier.
</ResponseField>

<ResponseField name="track" type="object">
  Information about the track being played.
</ResponseField>

<ResponseField name="startedAt" type="string">
  ISO 8601 timestamp when hold music started.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.talkos.io/api/features/hold-music/call_abc123/start \

  -d '{
    "trackId": "track_jazz_003",
    "volume": 70,
    "loop": true,
    "announcePosition": true,
    "estimatedWaitTime": 120
  }'
```
