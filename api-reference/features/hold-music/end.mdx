---
title: 'End Hold Music'
api: 'POST https://api.talkos.io/api/features/hold-music/{callUuid}/end'
description: 'Stop playing hold music on a call'
---

Stop playing hold music on a call, typically when taking the caller off hold to connect them with an agent.

## Path Parameters

<ParamField path="callUuid" type="string" required>
  The unique identifier of the call.
</ParamField>

## Request Body

<ParamField body="reason" type="string">
  Reason for ending the hold: `agent_connected`, `transfer_complete`, `caller_hung_up`, `timeout`, `manual`.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether hold music was stopped successfully.
</ResponseField>

<ResponseField name="holdId" type="string">
  The hold session identifier that was ended.
</ResponseField>

<ResponseField name="callUuid" type="string">
  The call's unique identifier.
</ResponseField>

<ResponseField name="duration" type="number">
  Total hold duration in seconds.
</ResponseField>

<ResponseField name="endedAt" type="string">
  ISO 8601 timestamp when hold music ended.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.talkos.io/api/features/hold-music/call_abc123/end \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "reason": "agent_connected"
  }'
```
