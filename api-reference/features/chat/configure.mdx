# Configure Chat

Configure the chat feature settings for your tenant, including appearance, behavior, automated responses, and integration preferences.

## Endpoint

```
POST /api/features/chat/configure
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="enabled" type="boolean" default={true}>
  Enable or disable chat functionality
</ParamField>

<ParamField body="autoGreeting" type="object">
  Automatic greeting configuration
</ParamField>

<ParamField body="autoGreeting.enabled" type="boolean" default={true}>
  Enable automatic greeting messages
</ParamField>

<ParamField body="autoGreeting.message" type="string">
  Custom greeting message text
</ParamField>

<ParamField body="autoGreeting.delayMs" type="integer" default={1000}>
  Delay before showing greeting in milliseconds
</ParamField>

<ParamField body="typingIndicator" type="object">
  Typing indicator settings
</ParamField>

<ParamField body="typingIndicator.showAgent" type="boolean" default={true}>
  Show when agent is typing
</ParamField>

<ParamField body="typingIndicator.showCustomer" type="boolean" default={true}>
  Show when customer is typing
</ParamField>

<ParamField body="readReceipts" type="boolean" default={true}>
  Enable read receipts for messages
</ParamField>

<ParamField body="maxMessageLength" type="integer" default={2000}>
  Maximum characters per message
</ParamField>

<ParamField body="fileSharing" type="object">
  File sharing configuration
</ParamField>

<ParamField body="offlineMode" type="object">
  Behavior when agents are offline
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the configuration was saved successfully
</ResponseField>

<ResponseField name="configId" type="string">
  Unique identifier for this configuration
</ResponseField>

<ResponseField name="settings" type="object">
  The complete configuration object as saved
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp when configuration was updated
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `INVALID_FILE_TYPE` | One or more file types are not supported |
| `MAX_SIZE_EXCEEDED` | File size limit exceeds allowed maximum |
| `INVALID_MESSAGE_LENGTH` | Message length limit is out of range |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
