# Add Canned Response

Create a new pre-defined canned response that agents can use during chat sessions for quick, consistent replies to common questions.

## Endpoint

```
POST /api/features/chat/canned-responses
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="shortcut" type="string" required>
  Keyboard shortcut to trigger the response (e.g., `/greeting`). Must start with `/`.
</ParamField>

<ParamField body="title" type="string" required>
  Display title for the canned response
</ParamField>

<ParamField body="content" type="string" required>
  The response message content. Supports `{{variable}}` placeholders.
</ParamField>

<ParamField body="category" type="string" required>
  Category for organizing responses
</ParamField>

<ParamField body="scope" type="string" default="global">
  Scope of the response: `global` (all agents) or `agent` (specific agent)
</ParamField>

<ParamField body="agentId" type="string">
  Agent ID if scope is `agent`
</ParamField>

<ParamField body="variables" type="array">
  Array of variable definitions for placeholders in content
</ParamField>

<ParamField body="tags" type="array">
  Tags for additional categorization and search
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the canned response was created successfully
</ResponseField>

<ResponseField name="response" type="object">
  The created canned response object
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `DUPLICATE_SHORTCUT` | A response with this shortcut already exists |
| `INVALID_SHORTCUT` | Shortcut must start with `/` and contain only alphanumeric characters |
| `CONTENT_TOO_LONG` | Content exceeds maximum length |
| `INVALID_VARIABLE` | Variable definition is malformed |
| `AGENT_NOT_FOUND` | Agent ID is invalid (when scope is `agent`) |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
