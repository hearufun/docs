# Send Typing Indicator

Send a typing indicator event to notify other participants that someone is currently typing a message. This creates a real-time, interactive chat experience.

## Endpoint

```
POST /api/features/chat/:callUuid/typing
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="callUuid" type="string" required>
  Unique identifier of the call with an active chat session
</ParamField>

## Request Body

<ParamField body="senderId" type="string" required>
  ID of the user who is typing
</ParamField>

<ParamField body="senderType" type="string" required>
  Type of sender: `agent`, `customer`, `bot`
</ParamField>

<ParamField body="isTyping" type="boolean" required>
  Whether the user is currently typing (`true`) or stopped typing (`false`)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the typing indicator was sent successfully
</ResponseField>

<ResponseField name="chatId" type="string">
  ID of the chat session
</ResponseField>

<ResponseField name="senderId" type="string">
  ID of the typing user
</ResponseField>

<ResponseField name="isTyping" type="boolean">
  Current typing status
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO 8601 timestamp of the event
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `CHAT_NOT_FOUND` | No active chat session found for this call |
| `CHAT_CLOSED` | The chat session has been closed |
| `TYPING_DISABLED` | Typing indicators are disabled in settings |
| `UNAUTHORIZED` | Invalid API key or tenant ID |

## Best Practices

- Send `isTyping: true` when the user starts typing
- Send `isTyping: false` when the user stops typing or sends a message
- Implement a debounce mechanism to avoid excessive API calls
- Typing indicators automatically expire after 5 seconds if not refreshed
