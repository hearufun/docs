# Get Chat History

Retrieve the complete message history for a chat session associated with a call. Returns all messages in chronological order with sender information and delivery status.

## Endpoint

```
GET /api/features/chat/:callUuid/history
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="callUuid" type="string" required>
  Unique identifier of the call
</ParamField>

## Query Parameters

<ParamField query="limit" type="integer" default={50}>
  Maximum number of messages to return (1-200)
</ParamField>

<ParamField query="before" type="string">
  Return messages before this message ID (for pagination)
</ParamField>

<ParamField query="after" type="string">
  Return messages after this message ID
</ParamField>

<ParamField query="includeMetadata" type="boolean" default={false}>
  Include message metadata in response
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the history was retrieved successfully
</ResponseField>

<ResponseField name="chatId" type="string">
  ID of the chat session
</ResponseField>

<ResponseField name="callUuid" type="string">
  Associated call UUID
</ResponseField>

<ResponseField name="messages" type="array">
  Array of chat messages in chronological order
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination information
</ResponseField>

<ResponseField name="participants" type="array">
  List of chat participants
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/features/chat/call-abc123-def456/history?limit=50&includeMetadata=true" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript JavaScript
const params = new URLSearchParams({
  limit: '50',
  includeMetadata: 'true'
});

const response = await fetch(`https://api.talkos.io/api/features/chat/call-abc123-def456/history?${params}`, {
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/features/chat/call-abc123-def456/history',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    },
    params={
        'limit': 50,
        'includeMetadata': True
    }
)

data = response.json()
```

## Error Codes

| Code | Description |
|------|-------------|
| `CHAT_NOT_FOUND` | No chat session exists for this call |
| `CALL_NOT_FOUND` | The specified call UUID does not exist |
| `INVALID_PAGINATION` | Invalid pagination parameters |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
