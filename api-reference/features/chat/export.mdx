# Export Chat Transcript

Export the complete chat transcript for a session in various formats. Useful for compliance, record-keeping, and sharing conversation history.

## Endpoint

```
GET /api/features/chat/:callUuid/export
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="callUuid" type="string" required>
  Unique identifier of the call
</ParamField>

## Query Parameters

<ParamField query="format" type="string" default="json">
  Export format: `json`, `txt`, `html`, `pdf`, `csv`
</ParamField>

<ParamField query="includeMetadata" type="boolean" default={false}>
  Include message metadata in export
</ParamField>

<ParamField query="includeTimestamps" type="boolean" default={true}>
  Include timestamps for each message
</ParamField>

<ParamField query="timezone" type="string" default="UTC">
  Timezone for timestamps (e.g., `America/New_York`)
</ParamField>

<ParamField query="redactPII" type="boolean" default={false}>
  Redact personally identifiable information
</ParamField>

## Response

For JSON format, returns structured data. For other formats, returns the file content with appropriate Content-Type header.

<ResponseField name="success" type="boolean">
  Whether the export was generated successfully
</ResponseField>

<ResponseField name="chatId" type="string">
  ID of the chat session
</ResponseField>

<ResponseField name="callUuid" type="string">
  Associated call UUID
</ResponseField>

<ResponseField name="format" type="string">
  Export format
</ResponseField>

<ResponseField name="transcript" type="object">
  The formatted transcript data (for JSON format)
</ResponseField>

<ResponseField name="downloadUrl" type="string">
  URL to download the exported file (for file formats)
</ResponseField>

<ResponseField name="expiresAt" type="string">
  ISO 8601 timestamp when download URL expires
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `CHAT_NOT_FOUND` | No chat session exists for this call |
| `INVALID_FORMAT` | The specified export format is not supported |
| `EXPORT_FAILED` | Export generation failed |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
