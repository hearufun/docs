# Get Chat Statistics

Retrieve comprehensive chat analytics including response times, satisfaction scores, volume metrics, and agent performance data.

## Endpoint

```
GET /api/features/chat/stats
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Query Parameters

<ParamField query="startDate" type="string" required>
  Start date for the statistics period (ISO 8601 format)
</ParamField>

<ParamField query="endDate" type="string" required>
  End date for the statistics period (ISO 8601 format)
</ParamField>

<ParamField query="groupBy" type="string" default="day">
  Time grouping: `hour`, `day`, `week`, `month`
</ParamField>

<ParamField query="agentId" type="string">
  Filter by specific agent ID
</ParamField>

<ParamField query="channel" type="string">
  Filter by channel: `web`, `sms`, `email`, `whatsapp`, `all`
</ParamField>

<ParamField query="includeAgentBreakdown" type="boolean" default={false}>
  Include per-agent statistics
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the statistics were retrieved successfully
</ResponseField>

<ResponseField name="period" type="object">
  The date range for the statistics
</ResponseField>

<ResponseField name="summary" type="object">
  Overall summary statistics
</ResponseField>

<ResponseField name="timeSeries" type="array">
  Time-grouped statistics data points
</ResponseField>

<ResponseField name="agentBreakdown" type="array">
  Per-agent statistics (if requested)
</ResponseField>

<ResponseField name="topCannedResponses" type="array">
  Most used canned responses
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/features/chat/stats?startDate=2026-02-01&endDate=2026-02-18&groupBy=day&includeAgentBreakdown=true" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript JavaScript
const params = new URLSearchParams({
  startDate: '2026-02-01',
  endDate: '2026-02-18',
  groupBy: 'day',
  includeAgentBreakdown: 'true'
});

const response = await fetch(`https://api.talkos.io/api/features/chat/stats?${params}`, {
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/features/chat/stats',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    },
    params={
        'startDate': '2026-02-01',
        'endDate': '2026-02-18',
        'groupBy': 'day',
        'includeAgentBreakdown': True
    }
)

data = response.json()
```

## Error Codes

| Code | Description |
|------|-------------|
| `INVALID_DATE_RANGE` | Start date must be before end date |
| `DATE_RANGE_TOO_LARGE` | Date range exceeds maximum allowed (90 days) |
| `AGENT_NOT_FOUND` | The specified agent ID is invalid |
| `INVALID_GROUP_BY` | Invalid groupBy parameter |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
