# Mark Messages as Read

Mark one or more messages as read by a specific participant. This updates the read status and triggers read receipts for other participants if enabled.

## Endpoint

```
POST /api/features/chat/:callUuid/read
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="callUuid" type="string" required>
  Unique identifier of the call with an active chat session
</ParamField>

## Request Body

<ParamField body="readerId" type="string" required>
  ID of the participant marking messages as read
</ParamField>

<ParamField body="readerType" type="string" required>
  Type of reader: `agent`, `customer`
</ParamField>

<ParamField body="messageIds" type="array">
  Specific message IDs to mark as read. If omitted, all unread messages are marked as read.
</ParamField>

<ParamField body="upToMessageId" type="string">
  Mark all messages up to and including this message ID as read
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the messages were marked as read successfully
</ResponseField>

<ResponseField name="chatId" type="string">
  ID of the chat session
</ResponseField>

<ResponseField name="readerId" type="string">
  ID of the reader
</ResponseField>

<ResponseField name="messagesRead" type="integer">
  Number of messages marked as read
</ResponseField>

<ResponseField name="lastReadMessageId" type="string">
  ID of the most recent message marked as read
</ResponseField>

<ResponseField name="readAt" type="string">
  ISO 8601 timestamp when messages were marked as read
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `CHAT_NOT_FOUND` | No active chat session found for this call |
| `MESSAGE_NOT_FOUND` | One or more message IDs do not exist |
| `RECEIPTS_DISABLED` | Read receipts are disabled in settings |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
