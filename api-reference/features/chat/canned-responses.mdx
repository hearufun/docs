# Get Canned Responses

Retrieve the list of pre-defined canned responses available for agents to use during chat sessions. Canned responses enable quick, consistent replies to common questions.

## Endpoint

```
GET /api/features/chat/canned-responses
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Query Parameters

<ParamField query="category" type="string">
  Filter by category
</ParamField>

<ParamField query="search" type="string">
  Search in shortcut and content
</ParamField>

<ParamField query="agentId" type="string">
  Include agent-specific responses
</ParamField>

<ParamField query="includeGlobal" type="boolean" default={true}>
  Include global responses
</ParamField>

<ParamField query="limit" type="integer" default={50}>
  Maximum responses to return
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the responses were retrieved successfully
</ResponseField>

<ResponseField name="responses" type="array">
  Array of canned responses
</ResponseField>

<ResponseField name="categories" type="array">
  List of available categories
</ResponseField>

<ResponseField name="totalCount" type="integer">
  Total number of canned responses
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/features/chat/canned-responses?category=billing&includeGlobal=true" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript JavaScript
const params = new URLSearchParams({
  category: 'billing',
  includeGlobal: 'true'
});

const response = await fetch(`https://api.talkos.io/api/features/chat/canned-responses?${params}`, {
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/features/chat/canned-responses',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    },
    params={
        'category': 'billing',
        'includeGlobal': True
    }
)

data = response.json()
```

## Error Codes

| Code | Description |
|------|-------------|
| `INVALID_CATEGORY` | The specified category does not exist |
| `AGENT_NOT_FOUND` | The specified agent ID is invalid |
| `UNAUTHORIZED` | Invalid API key or tenant ID |

## Variable Substitution

Canned responses support variables using `{{variable_name}}` syntax. When sending a message with a canned response, replace these placeholders with actual values.
