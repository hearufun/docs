# Send Chat Message

Send a text message, file, or rich content in an active chat session. Messages can be sent by agents or programmatically through automation.

## Endpoint

```
POST /api/features/chat/:callUuid/message
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="callUuid" type="string" required>
  Unique identifier of the call with an active chat session
</ParamField>

## Request Body

<ParamField body="senderId" type="string" required>
  ID of the message sender (agent ID or `system`)
</ParamField>

<ParamField body="senderType" type="string" required>
  Type of sender: `agent`, `system`, `bot`
</ParamField>

<ParamField body="content" type="string" required>
  The message text content
</ParamField>

<ParamField body="contentType" type="string" default="text">
  Type of content: `text`, `html`, `markdown`, `file`, `card`
</ParamField>

<ParamField body="attachments" type="array">
  Array of file attachments
</ParamField>

<ParamField body="richCard" type="object">
  Rich card content with buttons and actions
</ParamField>

<ParamField body="metadata" type="object">
  Additional metadata for the message
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the message was sent successfully
</ResponseField>

<ResponseField name="messageId" type="string">
  Unique identifier for the sent message
</ResponseField>

<ResponseField name="chatId" type="string">
  ID of the chat session
</ResponseField>

<ResponseField name="senderId" type="string">
  ID of the message sender
</ResponseField>

<ResponseField name="senderType" type="string">
  Type of the sender
</ResponseField>

<ResponseField name="content" type="string">
  The message content
</ResponseField>

<ResponseField name="deliveredAt" type="string">
  ISO 8601 timestamp when the message was delivered
</ResponseField>

<ResponseField name="status" type="string">
  Delivery status: `sent`, `delivered`, `read`
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `CHAT_NOT_FOUND` | No active chat session found for this call |
| `CHAT_CLOSED` | The chat session has been closed |
| `MESSAGE_TOO_LONG` | Message exceeds maximum length |
| `INVALID_CONTENT_TYPE` | The content type is not supported |
| `ATTACHMENT_TOO_LARGE` | File attachment exceeds size limit |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
