# Get Recent Escalations

Retrieve recent escalation events across all calls for your tenant. Useful for monitoring escalation activity and supervisor dashboards.

## Endpoint

```
GET /api/features/escalation/recent/all
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Query Parameters

<ParamField query="limit" type="number">
  Maximum number of escalations to return (default: 50, max: 200)
</ParamField>

## Response

<ResponseField name="escalations" type="array">
  Array of recent escalation events
</ResponseField>

<ResponseField name="escalations[].callUuid" type="string">
  Call UUID associated with the escalation
</ResponseField>

<ResponseField name="escalations[].ruleId" type="string">
  ID of the rule that was triggered
</ResponseField>

<ResponseField name="escalations[].ruleName" type="string">
  Name of the triggered rule
</ResponseField>

<ResponseField name="escalations[].priority" type="string">
  Priority level: `low`, `medium`, `high`, or `critical`
</ResponseField>

<ResponseField name="escalations[].action" type="object">
  Action that was triggered
</ResponseField>

<ResponseField name="escalations[].triggeredAt" type="string">
  ISO 8601 timestamp when the escalation was triggered
</ResponseField>

<ResponseField name="escalations[].resolved" type="boolean">
  Whether the escalation has been resolved
</ResponseField>

<ResponseField name="escalations[].agentId" type="string">
  ID of the agent handling the call
</ResponseField>

<ResponseField name="escalations[].customerNumber" type="string">
  Customer phone number (masked)
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/features/escalation/recent/all?limit=25" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript JavaScript
const response = await fetch('https://api.talkos.io/api/features/escalation/recent/all?limit=25', {

    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(

    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    },
```
