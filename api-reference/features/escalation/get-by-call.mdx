# Get Call Escalations

Retrieve the complete escalation history for a specific call, including all triggered rules and actions taken.

## Endpoint

```
GET /api/features/escalation/:callUuid
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="callUuid" type="string" required>
  Unique identifier of the call
</ParamField>

## Response

<ResponseField name="callUuid" type="string">
  The call UUID
</ResponseField>

<ResponseField name="escalations" type="array">
  Array of escalation events for this call
</ResponseField>

<ResponseField name="escalations[].ruleId" type="string">
  ID of the rule that was triggered
</ResponseField>

<ResponseField name="escalations[].ruleName" type="string">
  Name of the triggered rule
</ResponseField>

<ResponseField name="escalations[].priority" type="string">
  Priority level: `low`, `medium`, `high`, or `critical`
</ResponseField>

<ResponseField name="escalations[].action" type="object">
  Action that was triggered
</ResponseField>

<ResponseField name="escalations[].action.type" type="string">
  Action type: `transfer`, `notify`, or `flag`
</ResponseField>

<ResponseField name="escalations[].action.target" type="string">
  Target of the action
</ResponseField>

<ResponseField name="escalations[].triggeredAt" type="string">
  ISO 8601 timestamp when the escalation was triggered
</ResponseField>

<ResponseField name="escalations[].context" type="object">
  Context data at the time of escalation
</ResponseField>

<ResponseField name="escalations[].resolved" type="boolean">
  Whether the escalation has been resolved
</ResponseField>

<ResponseField name="escalations[].resolvedAt" type="string">
  ISO 8601 timestamp when resolved (if applicable)
</ResponseField>
