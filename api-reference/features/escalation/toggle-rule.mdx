# Toggle Escalation Rule

Enable or disable an escalation rule. This works for both built-in and custom rules.

## Endpoint

```
PATCH /api/features/escalation/rules/:ruleId/toggle
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Path Parameters

<ParamField path="ruleId" type="string" required>
  Unique identifier of the rule to toggle
</ParamField>

## Request Body

<ParamField body="enabled" type="boolean" required>
  Whether to enable (`true`) or disable (`false`) the rule
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the toggle was successful
</ResponseField>

<ResponseField name="rule" type="object">
  The updated escalation rule
</ResponseField>

<ResponseField name="rule.id" type="string">
  Unique identifier for the rule
</ResponseField>

<ResponseField name="rule.name" type="string">
  Name of the rule
</ResponseField>

<ResponseField name="rule.enabled" type="boolean">
  Current enabled state of the rule
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH https://api.talkos.io/api/features/escalation/rules/rule_negative_sentiment_3x/toggle \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id" \
  -d '{
    "enabled": false
  }'
```

```javascript JavaScript
const response = await fetch('https://api.talkos.io/api/features/escalation/rules/rule_negative_sentiment_3x/toggle', {
  method: 'PATCH',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  },
  body: JSON.stringify({
    enabled: false
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.patch(
    'https://api.talkos.io/api/features/escalation/rules/rule_negative_sentiment_3x/toggle',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    },
    json={
        'enabled': False
    }
)

data = response.json()
```
