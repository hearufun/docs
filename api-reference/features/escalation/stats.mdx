# Get Escalation Stats

Retrieve comprehensive escalation statistics and analytics for your tenant within a specified time window.

## Endpoint

```
GET /api/features/escalation/stats
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Query Parameters

<ParamField query="timeWindow" type="number">
  Time window in milliseconds (default: 3600000 = 1 hour)
</ParamField>

## Response

<ResponseField name="totalEscalations" type="number">
  Total number of escalations in the time window
</ResponseField>

<ResponseField name="byPriority" type="object">
  Escalations grouped by priority level
</ResponseField>

<ResponseField name="byPriority.low" type="number">
  Count of low priority escalations
</ResponseField>

<ResponseField name="byPriority.medium" type="number">
  Count of medium priority escalations
</ResponseField>

<ResponseField name="byPriority.high" type="number">
  Count of high priority escalations
</ResponseField>

<ResponseField name="byPriority.critical" type="number">
  Count of critical priority escalations
</ResponseField>

<ResponseField name="byAction" type="object">
  Escalations grouped by action type
</ResponseField>

<ResponseField name="byAction.transfer" type="number">
  Count of transfer actions
</ResponseField>

<ResponseField name="byAction.notify" type="number">
  Count of notify actions
</ResponseField>

<ResponseField name="byAction.flag" type="number">
  Count of flag actions
</ResponseField>

<ResponseField name="topRules" type="array">
  Most frequently triggered rules
</ResponseField>

<ResponseField name="topRules[].ruleId" type="string">
  Rule identifier
</ResponseField>

<ResponseField name="topRules[].ruleName" type="string">
  Rule name
</ResponseField>

<ResponseField name="topRules[].count" type="number">
  Number of times triggered
</ResponseField>

<ResponseField name="resolutionRate" type="number">
  Percentage of escalations that have been resolved
</ResponseField>

<ResponseField name="averageResolutionTimeMs" type="number">
  Average time to resolve escalations in milliseconds
</ResponseField>

<ResponseField name="timeWindow" type="number">
  The time window used for these stats in milliseconds
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/features/escalation/stats?timeWindow=86400000" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript JavaScript
const response = await fetch('https://api.talkos.io/api/features/escalation/stats?timeWindow=86400000', {
  method: 'GET',
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/features/escalation/stats',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    },
    params={
        'timeWindow': 86400000  # 24 hours
    }
)

data = response.json()
```
