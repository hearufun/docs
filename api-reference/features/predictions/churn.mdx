# Predict Churn Risk

Predict the likelihood of customer churn based on call history, sentiment trends, resolution rates, and behavioral patterns. Identify at-risk customers before they leave.

## Endpoint

```
POST /api/features/predictions/churn
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="customerId" type="string" required>
  Unique identifier of the customer to assess
</ParamField>

<ParamField body="timeframesDays" type="integer" default={90}>
  Number of days to analyze for churn prediction
</ParamField>

<ParamField body="includeRecommendations" type="boolean" default={true}>
  Include retention recommendations
</ParamField>

<ParamField body="includeTimeline" type="boolean" default={false}>
  Include historical churn risk timeline
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the churn prediction was generated successfully
</ResponseField>

<ResponseField name="predictionId" type="string">
  Unique identifier for this prediction
</ResponseField>

<ResponseField name="customerId" type="string">
  The customer ID that was assessed
</ResponseField>

<ResponseField name="churnRisk" type="string">
  Overall churn risk level: `low`, `moderate`, `high`, `critical`
</ResponseField>

<ResponseField name="churnProbability" type="number">
  Probability of churn (0-1)
</ResponseField>

<ResponseField name="confidence" type="number">
  Confidence level in the prediction (0-1)
</ResponseField>

<ResponseField name="predictedChurnWindow" type="string">
  Estimated timeframe for potential churn
</ResponseField>

<ResponseField name="riskFactors" type="array">
  Factors contributing to churn risk
</ResponseField>

<ResponseField name="recommendations" type="array">
  Suggested retention actions
</ResponseField>

<ResponseField name="timeline" type="array">
  Historical churn risk over time (if requested)
</ResponseField>

<ResponseField name="assessedAt" type="string">
  ISO 8601 timestamp of the assessment
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `CUSTOMER_NOT_FOUND` | The specified customer ID does not exist |
| `INSUFFICIENT_HISTORY` | Not enough interaction history for prediction |
| `ENGINE_NOT_READY` | Prediction models are still training |
| `INVALID_TIMEFRAME` | The specified timeframe is invalid |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
