# Initialize Predictions

Initialize and configure the predictive analytics engine for your tenant. This sets up the machine learning models and data pipelines required for generating predictions about call outcomes, customer behavior, and operational metrics.

## Endpoint

```
POST /api/features/predictions/init
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="models" type="array">
  List of prediction models to enable: `churn`, `escalation`, `satisfaction`, `duration`, `conversion`
</ParamField>

<ParamField body="trainingDataDays" type="integer" default={30}>
  Number of days of historical data to use for initial model training
</ParamField>

<ParamField body="updateFrequency" type="string" default="daily">
  How often models should be retrained: `hourly`, `daily`, `weekly`
</ParamField>

<ParamField body="confidenceThreshold" type="number" default={0.7}>
  Minimum confidence score (0-1) for predictions to be surfaced
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the prediction engine was initialized successfully
</ResponseField>

<ResponseField name="engineId" type="string">
  Unique identifier for the prediction engine instance
</ResponseField>

<ResponseField name="models" type="array">
  List of enabled prediction models with their status
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the engine: `initializing`, `training`, `ready`
</ResponseField>

<ResponseField name="estimatedReadyTime" type="string">
  ISO 8601 timestamp when models are expected to be ready
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp when initialization was triggered
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `ENGINE_EXISTS` | Prediction engine already initialized for this tenant |
| `INVALID_MODEL` | One or more specified models are invalid |
| `INSUFFICIENT_DATA` | Not enough historical data for training |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
