# Record Prediction Event

Record events, outcomes, and feedback data to continuously improve prediction model accuracy. This endpoint accepts training data points that help the ML models learn from real-world results.

## Endpoint

```
POST /api/features/predictions/record
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="eventType" type="string" required>
  Type of event: `call_outcome`, `customer_action`, `agent_action`, `escalation`, `churn`, `satisfaction`
</ParamField>

<ParamField body="callUuid" type="string">
  UUID of the associated call (if applicable)
</ParamField>

<ParamField body="customerId" type="string">
  Unique identifier of the customer
</ParamField>

<ParamField body="outcome" type="string" required>
  The actual outcome that occurred: `positive`, `negative`, `neutral`
</ParamField>

<ParamField body="predictionId" type="string">
  ID of a previous prediction to provide feedback on
</ParamField>

<ParamField body="metadata" type="object">
  Additional contextual data relevant to the event
</ParamField>

<ParamField body="timestamp" type="string">
  ISO 8601 timestamp of when the event occurred (defaults to now)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the event was recorded successfully
</ResponseField>

<ResponseField name="eventId" type="string">
  Unique identifier for the recorded event
</ResponseField>

<ResponseField name="eventType" type="string">
  The type of event that was recorded
</ResponseField>

<ResponseField name="modelUpdated" type="boolean">
  Whether this event triggered a model update
</ResponseField>

<ResponseField name="recordedAt" type="string">
  ISO 8601 timestamp when the event was recorded
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `INVALID_EVENT_TYPE` | The specified event type is not supported |
| `PREDICTION_NOT_FOUND` | The referenced prediction ID does not exist |
| `ENGINE_NOT_INITIALIZED` | Prediction engine has not been initialized |
| `INVALID_OUTCOME` | The outcome value is not valid |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
