# Predict Escalation Risk

Analyze real-time signals and historical data to predict the likelihood of a call escalating to a supervisor, complaint, or negative outcome. Returns risk scores and recommended interventions.

## Endpoint

```
POST /api/features/predictions/escalation-risk
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="callUuid" type="string" required>
  UUID of the active call to assess
</ParamField>

<ParamField body="includeRealtime" type="boolean" default={true}>
  Include real-time sentiment and tone analysis
</ParamField>

<ParamField body="includeSuggestions" type="boolean" default={true}>
  Include de-escalation suggestions
</ParamField>

<ParamField body="sensitivityLevel" type="string" default="medium">
  Risk detection sensitivity: `low`, `medium`, `high`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the escalation risk was assessed successfully
</ResponseField>

<ResponseField name="predictionId" type="string">
  Unique identifier for this prediction
</ResponseField>

<ResponseField name="callUuid" type="string">
  The call UUID that was assessed
</ResponseField>

<ResponseField name="riskLevel" type="string">
  Overall risk level: `low`, `medium`, `high`, `critical`
</ResponseField>

<ResponseField name="riskScore" type="number">
  Numerical risk score from 0-100
</ResponseField>

<ResponseField name="confidence" type="number">
  Confidence level in the prediction (0-1)
</ResponseField>

<ResponseField name="riskFactors" type="array">
  Specific factors contributing to the risk score
</ResponseField>

<ResponseField name="realtimeSignals" type="object">
  Real-time analysis of current call signals
</ResponseField>

<ResponseField name="suggestions" type="array">
  Recommended actions to reduce escalation risk
</ResponseField>

<ResponseField name="assessedAt" type="string">
  ISO 8601 timestamp of the assessment
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `CALL_NOT_FOUND` | The specified call UUID does not exist |
| `CALL_NOT_ACTIVE` | The call has ended and cannot be assessed |
| `ENGINE_NOT_READY` | Prediction models are still training |
| `REALTIME_UNAVAILABLE` | Real-time analysis could not be performed |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
