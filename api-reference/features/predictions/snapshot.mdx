# Get Predictions Snapshot

Retrieve a comprehensive snapshot of all prediction metrics, model performance, and current insights for your tenant. Provides an overview of the predictive analytics system status and recent predictions.

## Endpoint

```
GET /api/features/predictions/snapshot
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Query Parameters

<ParamField query="includeModels" type="boolean" default={true}>
  Include detailed model performance metrics
</ParamField>

<ParamField query="includeInsights" type="boolean" default={true}>
  Include AI-generated insights and anomalies
</ParamField>

<ParamField query="includeTopPredictions" type="boolean" default={true}>
  Include top recent predictions by category
</ParamField>

<ParamField query="hoursBack" type="integer" default={24}>
  Number of hours to include in the snapshot
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the snapshot was retrieved successfully
</ResponseField>

<ResponseField name="snapshotId" type="string">
  Unique identifier for this snapshot
</ResponseField>

<ResponseField name="engineStatus" type="object">
  Current status of the prediction engine
</ResponseField>

<ResponseField name="models" type="array">
  Performance metrics for each prediction model
</ResponseField>

<ResponseField name="recentActivity" type="object">
  Summary of recent prediction activity
</ResponseField>

<ResponseField name="insights" type="array">
  AI-generated insights and detected anomalies
</ResponseField>

<ResponseField name="topPredictions" type="object">
  Top predictions by category (churn, escalation, etc.)
</ResponseField>

<ResponseField name="generatedAt" type="string">
  ISO 8601 timestamp when snapshot was generated
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/features/predictions/snapshot?includeModels=true&includeInsights=true&hoursBack=24" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript JavaScript
const params = new URLSearchParams({
  includeModels: 'true',
  includeInsights: 'true',
  includeTopPredictions: 'true',
  hoursBack: '24'
});

const response = await fetch(`https://api.talkos.io/api/features/predictions/snapshot?${params}`, {
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/features/predictions/snapshot',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    },
    params={
        'includeModels': True,
        'includeInsights': True,
        'includeTopPredictions': True,
        'hoursBack': 24
    }
)

data = response.json()
```

## Error Codes

| Code | Description |
|------|-------------|
| `ENGINE_NOT_INITIALIZED` | Prediction engine has not been set up |
| `ENGINE_UNHEALTHY` | Prediction engine is experiencing issues |
| `INVALID_HOURS_BACK` | The hoursBack parameter is out of range |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
