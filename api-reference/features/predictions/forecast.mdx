# Get Call Volume Forecast

Retrieve AI-powered call volume forecasts to optimize staffing, capacity planning, and resource allocation. Predictions are based on historical patterns, seasonality, and external factors.

## Endpoint

```
GET /api/features/predictions/forecast
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Query Parameters

<ParamField query="period" type="string" default="day">
  Forecast period: `hour`, `day`, `week`, `month`
</ParamField>

<ParamField query="horizon" type="integer" default={7}>
  Number of periods to forecast ahead
</ParamField>

<ParamField query="granularity" type="string" default="hourly">
  Detail level: `15min`, `30min`, `hourly`, `daily`
</ParamField>

<ParamField query="includeConfidence" type="boolean" default={true}>
  Include confidence intervals in results
</ParamField>

<ParamField query="callType" type="string">
  Filter by call type: `inbound`, `outbound`, `all`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the forecast was retrieved successfully
</ResponseField>

<ResponseField name="forecastId" type="string">
  Unique identifier for this forecast
</ResponseField>

<ResponseField name="period" type="string">
  The forecast period requested
</ResponseField>

<ResponseField name="horizon" type="integer">
  Number of periods forecasted
</ResponseField>

<ResponseField name="granularity" type="string">
  Detail level of the forecast
</ResponseField>

<ResponseField name="forecast" type="array">
  Array of forecast data points
</ResponseField>

<ResponseField name="summary" type="object">
  Summary statistics for the forecast period
</ResponseField>

<ResponseField name="modelAccuracy" type="number">
  Historical accuracy of the forecast model (0-1)
</ResponseField>

<ResponseField name="generatedAt" type="string">
  ISO 8601 timestamp when forecast was generated
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.talkos.io/api/features/predictions/forecast?period=day&horizon=7&granularity=hourly&includeConfidence=true" \
  -H "x-api-key: your-api-key" \
  -H "x-tenant-id: your-tenant-id"
```

```javascript JavaScript
const params = new URLSearchParams({
  period: 'day',
  horizon: '7',
  granularity: 'hourly',
  includeConfidence: 'true'
});

const response = await fetch(`https://api.talkos.io/api/features/predictions/forecast?${params}`, {
  headers: {
    'x-api-key': 'your-api-key',
    'x-tenant-id': 'your-tenant-id'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.talkos.io/api/features/predictions/forecast',
    headers={
        'x-api-key': 'your-api-key',
        'x-tenant-id': 'your-tenant-id'
    },
    params={
        'period': 'day',
        'horizon': 7,
        'granularity': 'hourly',
        'includeConfidence': True
    }
)

data = response.json()
```

## Error Codes

| Code | Description |
|------|-------------|
| `INSUFFICIENT_DATA` | Not enough historical data for forecasting |
| `INVALID_PERIOD` | The specified period is not supported |
| `INVALID_GRANULARITY` | The granularity is not valid for the period |
| `ENGINE_NOT_READY` | Prediction models are still training |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
