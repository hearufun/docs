# Predict Call Outcome

Generate real-time predictions for an active or upcoming call, including expected duration, satisfaction score, resolution likelihood, and recommended actions.

## Endpoint

```
POST /api/features/predictions/call
```

## Headers

<ParamField header="x-api-key" type="string" required>
  Your TalkOS API key for authentication
</ParamField>

<ParamField header="x-tenant-id" type="string" required>
  Your unique tenant identifier
</ParamField>

## Request Body

<ParamField body="callUuid" type="string">
  UUID of an active call to generate predictions for
</ParamField>

<ParamField body="customerId" type="string">
  Customer ID to generate pre-call predictions
</ParamField>

<ParamField body="phoneNumber" type="string">
  Phone number for caller identification (if no customerId)
</ParamField>

<ParamField body="agentId" type="string">
  Agent ID handling the call (for agent-specific predictions)
</ParamField>

<ParamField body="callType" type="string">
  Type of call: `inbound`, `outbound`, `transfer`
</ParamField>

<ParamField body="context" type="object">
  Additional context to improve prediction accuracy
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether predictions were generated successfully
</ResponseField>

<ResponseField name="predictionId" type="string">
  Unique identifier for this prediction set
</ResponseField>

<ResponseField name="predictions" type="object">
  Container for all prediction results
</ResponseField>

<ResponseField name="predictions.duration" type="object">
  Expected call duration prediction
</ResponseField>

<ResponseField name="predictions.satisfaction" type="object">
  Predicted customer satisfaction score
</ResponseField>

<ResponseField name="predictions.resolution" type="object">
  First-call resolution likelihood
</ResponseField>

<ResponseField name="predictions.escalation" type="object">
  Escalation probability
</ResponseField>

<ResponseField name="recommendations" type="array">
  AI-generated recommendations for handling the call
</ResponseField>

<ResponseField name="generatedAt" type="string">
  ISO 8601 timestamp when predictions were generated
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `CALL_NOT_FOUND` | The specified call UUID does not exist |
| `CUSTOMER_NOT_FOUND` | The customer ID could not be found |
| `ENGINE_NOT_READY` | Prediction models are still training |
| `INSUFFICIENT_DATA` | Not enough data to generate predictions |
| `UNAUTHORIZED` | Invalid API key or tenant ID |
